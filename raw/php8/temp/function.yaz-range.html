yaz_range

(PHP 4 >= 4.0.1, PECL yaz >= 0.9.0)

yaz_range — Specifies a range of records to retrieve



说明

yaz_range(resource $id, int $start, int $number): void

Specifies a range of records to retrieve.

This function should be called before {@link yaz_search()} or {@link yaz_present()}.



参数

id

The connection resource returned by {@link yaz_connect()}.

start

Specifies the position of the first record to be retrieved. The records numbers goes from 1 to {@link yaz_hits()}.

number

Specifies the number of records to be retrieved.



返回值

没有返回值。
