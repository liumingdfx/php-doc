GearmanJob::data

(PECL gearman <= 0.5.0)

GearmanJob::data — Send data for a running job (deprecated)



说明

public GearmanJob::data(string $data): bool

Sends data to the job server (and any listening clients) for this job.

注意:

This method has been replaced by {@link GearmanJob::sendData()} in the 0.6.0 release of the Gearman extension.



参数

data

Arbitrary serialized data.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link GearmanJob::workload()} - Get workload
  • {@link GearmanTask::data()} - Get data returned for a task
