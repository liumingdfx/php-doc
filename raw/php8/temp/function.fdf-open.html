fdf_open

(PHP 4, PHP 5 < 5.3.0, PECL fdf SVN)

fdf_open — Open a FDF document



说明

fdf_open(string $filename): resource

Opens a file with form data.

You can also use {@link fdf_open_string()} to process the results of a PDF form POST request.



参数

filename

Path to the FDF file. This file must contain the data as returned from a PDF form or created using {@link fdf_create()} and {@link fdf_save()}.



返回值

Returns a FDF document handle, or false on error.



示例

示例 #1 Accessing the form data

<?php
// Save the FDF data into a temp file
$fdffp = fopen("test.fdf", "w");
fwrite($fdffp, $HTTP_FDF_DATA, strlen($HTTP_FDF_DATA));
fclose($fdffp);

// Open temp file and evaluate data
$fdf = fdf_open("test.fdf");
// ...
fdf_close($fdf);
?>
