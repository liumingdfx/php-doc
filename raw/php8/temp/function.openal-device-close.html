openal_device_close

(PECL openal >= 0.1.0)

openal_device_close — Close an OpenAL device



说明

openal_device_close(resource $device): bool


参数

device
An Open AL(Device) resource (previously created by {@link openal_device_open()}) to be closed.


返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link openal_device_open()} - Initialize the OpenAL audio layer
