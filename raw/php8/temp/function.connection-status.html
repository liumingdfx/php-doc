connection_status

(PHP 4, PHP 5, PHP 7, PHP 8)

connection_status — 返回连接的状态位



说明

connection_status(): int

获得当前连接的状态位。



参数

此函数没有参数。


返回值

获得当前连接的状态位, 可以用于与 CONNECTION_XXX 定义的常量来确定连接状态。



参见

  • {@link connection_aborted()} - 检查客户端是否已经断开
  • {@link ignore_user_abort()} - 设置客户端断开连接时是否中断脚本的执行
  • 查看连接处理了解PHP处理连接的详情。
