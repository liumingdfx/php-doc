The DOMChildNode interface

(PHP 8)

接口摘要

interface DOMChildNode {
// 方法
public after(DOMNode|string ...$nodes): void
public before(DOMNode|string ...$nodes): void
public remove(): void
public replaceWith(DOMNode|string ...$nodes): void
}

目录

  • {@link DOMChildNode::after} — Adds nodes after the node
  • {@link DOMChildNode::before} — Adds nodes before the node
  • {@link DOMChildNode::remove} — Removes the node
  • {@link DOMChildNode::replaceWith} — Replaces the node with new nodes
