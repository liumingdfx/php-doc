GearmanTask::function

(PECL gearman <= 0.5.0)

GearmanTask::function — Get associated function name (deprecated)



说明

public GearmanTask::function(): string

Returns the name of the function this task is associated with, i.e., the function the Gearman worker calls.

注意:

This method has been replaced by {@link GearmanTask::functionName()} in the 0.6.0 release of the Gearman extension.



参数

此函数没有参数。


返回值

A function name.
