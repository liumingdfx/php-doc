DOMChildNode::remove

(PHP 8)

DOMChildNode::remove — Removes the node



说明

public DOMChildNode::remove(): void

Removes the node.



参数

此函数没有参数。


返回值

没有返回值。



参见

  • {@link DOMChildNode::after()} - Adds nodes after the node
  • {@link DOMChildNode::before()} - Adds nodes before the node
  • {@link DOMChildNode::replaceWith()} - Replaces the node with new nodes
  • {@link DOMNode::removeChild()} - Removes child from list of children
