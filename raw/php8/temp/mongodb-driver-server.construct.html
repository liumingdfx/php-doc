MongoDB\Driver\Server::__construct

(mongodb >=1.0.0)

MongoDB\Driver\Server::__construct — Create a new Server (not used)



说明

final private MongoDB\Driver\Server::__construct()

MongoDB\Driver\Server objects are created internally by MongoDB\Driver\Manager when a database connection is established and may be returned by {@link MongoDB\Driver\Manager::getServers()} and {@link MongoDB\Driver\Manager::selectServer()}.



参数

此函数没有参数。


参见

  • {@link MongoDB\Driver\Manager::getServers()} - Return the servers to which this manager is connected
  • {@link MongoDB\Driver\Manager::selectServer()} - Select a server matching a read preference
