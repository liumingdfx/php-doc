XMLReader::close

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

XMLReader::close — Close the XMLReader input



说明

public XMLReader::close(): true

Closes the input the XMLReader object is currently parsing.



参数

此函数没有参数。


返回值

总是返回 true。



参见

  • {@link XMLReader::open()} - Set the URI containing the XML to parse
  • {@link XMLReader::xml()} - Set the data containing the XML to parse
