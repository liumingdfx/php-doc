gztell

(PHP 4, PHP 5, PHP 7, PHP 8)

gztell — Tell gz-file pointer read/write position



说明

gztell(resource $stream): int|false

Gets the position of the given file pointer; i.e., its offset into the uncompressed file stream.



参数

stream

The gz-file pointer. It must be valid, and must point to a file successfully opened by {@link gzopen()}.



返回值

The position of the file pointer or false if an error occurs.



参见

  • {@link gzopen()} - Open gz-file
  • {@link gzseek()} - Seek on a gz-file pointer
  • {@link gzrewind()} - Rewind the position of a gz-file pointer
