EvPrepare::__construct

(PECL ev >= 0.2.0)

EvPrepare::__construct — Constructs EvPrepare watcher object



说明

public EvPrepare::__construct( string $callback , string $data = ?, string $priority = ?)

Constructs EvPrepare watcher object. And starts the watcher automatically. If need a stopped watcher consider using {@link EvPrepare::createStopped()}



参数

callback
See Watcher callbacks .
data

Custom data associated with the watcher.

priority

Watcher priority
