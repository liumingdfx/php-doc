The Swoole\Buffer class

(PECL swoole >= 1.9.0)

简介

类摘要

class Swoole\Buffer {
// 方法
public append(string $data): int
public clear(): void
public __destruct(): void
public expand(int $size): int
public read(int $offset, int $length): string
public recycle(): void
public substr(int $offset, int $length = ?, bool $remove = ?): string

}

目录

  • {@link Swoole\Buffer::append} — Append the string or binary data at the end of the memory buffer and return the new size of memory allocated.
  • {@link Swoole\Buffer::clear} — Reset the memory buffer.
  • {@link Swoole\Buffer::__construct} — Fixed size memory blocks allocation.
  • {@link Swoole\Buffer::__destruct} — Destruct the Swoole memory buffer.
  • {@link Swoole\Buffer::expand} — Expand the size of memory buffer.
  • {@link Swoole\Buffer::read} — Read data from the memory buffer based on offset and length.
  • {@link Swoole\Buffer::recycle} — Release the memory to OS which is not used by the memory buffer.
  • {@link Swoole\Buffer::substr} — Read data from the memory buffer based on offset and length. Or remove data from the memory buffer.
  • {@link Swoole\Buffer::__toString} — Get the string value of the memory buffer.
  • {@link Swoole\Buffer::write} — Write data to the memory buffer. The memory allocated for the buffer will not be changed.
