SolrParams::toString

(PECL solr >= 0.9.2)

SolrParams::toString — Returns all the name-value pair parameters in the object



说明

final public SolrParams::toString(bool $url_encode = false): string

Returns all the name-value pair parameters in the object



参数

url_encode
