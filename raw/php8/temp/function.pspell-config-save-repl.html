pspell_config_save_repl

(PHP 4 >= 4.0.2, PHP 5, PHP 7, PHP 8)

pspell_config_save_repl — Determine whether to save a replacement pairs list along with the wordlist



说明

pspell_config_save_repl(PSpell\Config $config, bool $save): bool

pspell_config_save_repl() determines whether {@link pspell_save_wordlist()} will save the replacement pairs along with the wordlist. Usually there is no need to use this function because if {@link pspell_config_repl()} is used, the replacement pairs will be saved by {@link pspell_save_wordlist()} anyway, and if it is not, the replacement pairs will not be saved.

pspell_config_save_repl() should be used on a config before calling {@link pspell_new_config()}.



参数

config

An PSpell\Config instance.

save

true if replacement pairs should be saved, false otherwise.



返回值

成功时返回 true， 或者在失败时返回 false。



更新日志

版本 说明

注释

注意:

This function will not work unless you have pspell .11.2 and aspell .32.5 or later.
