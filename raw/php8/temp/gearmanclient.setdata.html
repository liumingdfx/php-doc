GearmanClient::setData

(PECL gearman <= 0.5.0)

GearmanClient::setData — Set application data (deprecated)



说明

public GearmanClient::setData(string $data): bool

Sets some arbitrary application data that can later be retrieved by {@link GearmanClient::data()}.

注意:

This method has been replaced by GearmanCient::setContext() in the 0.6.0 release of the Gearman extension.



参数

data


返回值

Always returns true.



参见

  • {@link GearmanClient::data()} - Get the application data (deprecated)
