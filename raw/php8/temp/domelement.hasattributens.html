DOMElement::hasAttributeNS

(PHP 5, PHP 7, PHP 8)

DOMElement::hasAttributeNS — Checks to see if attribute exists



说明

public DOMElement::hasAttributeNS(?string $namespace, string $localName): bool

Indicates whether attribute in namespace namespace named localName exists as a member of the element.



参数

namespace

The namespace URI.

localName

The local name.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link DOMElement::hasAttribute()} - Checks to see if attribute exists
  • {@link DOMElement::getAttributeNS()} - Returns value of attribute
  • {@link DOMElement::setAttributeNS()} - Adds new attribute
  • {@link DOMElement::removeAttributeNS()} - Removes attribute
