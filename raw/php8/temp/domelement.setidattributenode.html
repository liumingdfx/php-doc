DOMElement::setIdAttributeNode

(PHP 5, PHP 7, PHP 8)

DOMElement::setIdAttributeNode — Declares the attribute specified by node to be of type ID



说明

public DOMElement::setIdAttributeNode(DOMAttr $attr, bool $isId): void

Declares the attribute specified by attr to be of type ID.



参数

attr

The attribute node.

isId

Set it to true if you want name to be of type ID, false otherwise.



返回值

没有返回值。



错误／异常

DOM_NO_MODIFICATION_ALLOWED_ERR

Raised if the node is readonly.

DOM_NOT_FOUND

Raised if name is not an attribute of this element.



参见

  • {@link DOMDocument::getElementById()} - Searches for an element with a certain id
  • {@link DOMElement::setIdAttribute()} - Declares the attribute specified by name to be of type ID
  • {@link DOMElement::setIdAttributeNS()} - Declares the attribute specified by local name and namespace URI to be of type ID
