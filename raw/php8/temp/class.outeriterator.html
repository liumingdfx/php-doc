The OuterIterator interface

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

简介

Classes implementing OuterIterator can be used to iterate over iterators.

接口摘要

interface OuterIterator extends Iterator {
// 方法
public getInnerIterator(): ?Iterator
// 继承的方法
public {@link Iterator::current}(): mixed
public {@link Iterator::key}(): mixed
public {@link Iterator::next}(): void
public {@link Iterator::rewind}(): void
public {@link Iterator::valid}(): bool
}
