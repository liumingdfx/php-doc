zend_version

(PHP 4, PHP 5, PHP 7, PHP 8)

zend_version — 获取当前 Zend 引擎的版本



说明

zend_version(): string

获取当前运行的 Zend 引擎的版本字符串。



参数

此函数没有参数。


返回值

获取 Zend 引擎的版本数字的字符串。



示例

示例 #1 zend_version() 示例

<?php
echo "Zend engine version: " . zend_version();
?>

以上示例的输出类似于：

Zend engine version: 2.2.0
