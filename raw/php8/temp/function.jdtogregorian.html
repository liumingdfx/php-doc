jdtogregorian

(PHP 4, PHP 5, PHP 7, PHP 8)

jdtogregorian — 将儒略日数转换为公历日期



说明

jdtogregorian(int $julian_day): string

将儒略日数转换为“月/日/年”格式，string 类型的公历日期。



参数

julian_day

int 类型的儒略日数



返回值

格式为 “月/日/年”，string 类型的公历日期



参见

  • {@link gregoriantojd()} - 将公历日期转为儒略日数
  • {@link cal_from_jd()} - 从儒略日数转换为支持的历法
