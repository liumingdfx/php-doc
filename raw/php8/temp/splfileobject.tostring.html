SplFileObject::__toString

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

SplFileObject::__toString — Returns the current line as a string



说明

public SplFileObject::__toString(): string

This method will return the current line as a string.



参数

此函数没有参数。


返回值

Returns the current line as a string.



更新日志

版本 说明
8.1.14, 8.2.1 Changed from an alias of {@link SplFileObject::fgets()} to an implementation of {@link SplFileObject::current()} which returns a CSV string when the SplFileObject::READ_CSV flag is set.
7.2.19, 7.3.6 Changed from an alias of {@link SplFileObject::current()} to an alias of {@link SplFileObject::fgets()}.
