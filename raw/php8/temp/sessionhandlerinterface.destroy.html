SessionHandlerInterface::destroy

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

SessionHandlerInterface::destroy — Destroy a session



说明

public SessionHandlerInterface::destroy(string $id): bool

Destroys a session. Called by {@link session_regenerate_id()} (with $destroy = true), {@link session_destroy()} and when {@link session_decode()} fails.



参数

id

The session ID being destroyed.



返回值

会话存储的返回值（通常成功返回 0，失败返回 1）。
