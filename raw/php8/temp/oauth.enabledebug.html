OAuth::enableDebug

(PECL OAuth >= 0.99.3)

OAuth::enableDebug — 启用详细调试



说明

public OAuth::enableDebug(): bool

打开用于调试的详细请求信息，调试信息存储在 debugInfo 成员中。或者，可以设置 debug 成员为一个非 false 值来打开启用调试。



参数

此函数没有参数。


返回值

true



更新日志

版本 说明
PECL oauth 0.99.8 增加了 debug 和 debugInfo 成员


参见

  • {@link OAuth::disableDebug()} - 关闭详细的调试
