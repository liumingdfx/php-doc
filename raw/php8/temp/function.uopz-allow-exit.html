uopz_allow_exit

(PECL uopz 5, PECL uopz 6, PECL uopz 7)

uopz_allow_exit — Allows control over disabled exit opcode



说明

uopz_allow_exit(bool $allow): void

By default uopz disables the exit opcode, so {@link exit()} calls are practically ignored. uopz_allow_exit() allows to control this behavior.



参数

allow
Whether to allow the execution of exit opcodes or not.


返回值

没有返回值。



示例

示例 #1 uopz_allow_exit() example

<?php
exit(1);
echo 1;
uopz_allow_exit(true);
exit(2);
echo 2;
?>

以上示例会输出：

1


注释

警告

OPcache optimizes away dead code after unconditional exit.
