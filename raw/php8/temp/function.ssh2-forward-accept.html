ssh2_forward_accept

(PECL ssh2 >= 0.9.0)

ssh2_forward_accept — Accept a connection created by a listener



说明

ssh2_forward_accept(resource $listener): resource|false

Accepts a connection created by a listener.

警告

本函数还未编写文档，仅有参数列表。



参数

desc
An SSH2 Listener resource, obtained from a call to {@link ssh2_forward_listen()}.


返回值

Returns a stream resource, 或者在失败时返回 false.
