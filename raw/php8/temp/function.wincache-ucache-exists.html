wincache_ucache_exists

(PECL wincache >= 1.1.0)

wincache_ucache_exists — Checks if a variable exists in the user cache



说明

wincache_ucache_exists(string $key): bool

Checks if a variable with the key exists in the user cache or not.



参数

key

The key that was used to store the variable in the cache. key is case sensitive.



返回值

Returns true if variable with the key exitsts, otherwise returns false.



示例

示例 #1 Using wincache_ucache_exists()

<?php
if (!wincache_ucache_exists('green'))
wincache_ucache_set('green', 1);
var_dump(wincache_ucache_exists('green'));
?>

以上示例会输出：

bool(true)


参见

  • {@link wincache_ucache_set()} - Adds a variable in user cache and overwrites a variable if it already exists in the cache
  • {@link wincache_ucache_add()} - Adds a variable in user cache only if variable does not already exist in the cache
  • {@link wincache_ucache_get()} - Gets a variable stored in the user cache
  • {@link wincache_ucache_clear()} - Deletes entire content of the user cache
  • {@link wincache_ucache_delete()} - Deletes variables from the user cache
  • {@link wincache_ucache_meminfo()} - Retrieves information about user cache memory usage
  • {@link wincache_ucache_info()} - Retrieves information about data stored in the user cache
