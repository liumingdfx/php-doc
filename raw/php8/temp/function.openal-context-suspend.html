openal_context_suspend

(PECL openal >= 0.1.0)

openal_context_suspend — Suspend the specified context



说明

openal_context_suspend(resource $context): bool


参数

context
An Open AL(Context) resource (previously created by {@link openal_context_create()}).


返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link openal_context_create()} - Create an audio processing context
  • {@link openal_context_current()} - Make the specified context current
  • {@link openal_context_process()} - Process the specified context
