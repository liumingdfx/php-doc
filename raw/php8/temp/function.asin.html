asin

(PHP 4, PHP 5, PHP 7, PHP 8)

asin — 反正弦



说明

asin(float $num): float

返回 num 的反正弦值，单位是弧度。asin() 是 {@link sin()} 的反函数，它的意思是对于 asin() 的定义域中的每个 num 值，$num == sin(asin($num))。



参数

num

待处理的参数



返回值

num 的反正弦弧度。



参见

  • {@link sin()} - 正弦
  • {@link asinh()} - 反双曲正弦
  • {@link acos()} - 反余弦
  • {@link atan()} - 反正切
