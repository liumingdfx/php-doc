JsonException 类

(PHP 7 >= 7.3.0, PHP 8)

简介

如果为 {@link json_encode()} 或 {@link json_decode()} 设置了 JSON_THROW_ON_ERROR 选项，则会抛出异常。 code 包含错误类型，关于可能的值请参见 {@link json_last_error()}。
