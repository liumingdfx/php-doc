imap_bodystruct

(PHP 4, PHP 5, PHP 7, PHP 8)

imap_bodystruct — Read the structure of a specified body section of a specific message



说明

imap_bodystruct(IMAP\Connection $imap, int $message_num, string $section): stdClass|false

Read the structure of a specified body section of a specific message.



参数

imap

IMAP\Connection 实例。

message_num

The message number

section

The body section to read



返回值

Returns the information in an object, 或者在失败时返回 false. For a detailed description of the object structure and properties see {@link imap_fetchstructure()}.



更新日志

版本 说明
8.1.0 现在 imap 参数接受 IMAP\Connection 实例，之前接受有效的 imap resource。
