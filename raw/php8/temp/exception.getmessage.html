Exception::getMessage

(PHP 5, PHP 7, PHP 8)

Exception::getMessage — 获取异常消息内容



说明

final public Exception::getMessage(): string

返回异常消息内容。



参数

此函数没有参数。


返回值

返回字符串类型的异常消息内容。



示例

示例 #1 Exception::getMessage()示例

<?php
try {
throw new Exception("Some error message");
} catch(Exception $e) {
echo $e->getMessage();
}
?>

以上示例的输出类似于：

Some error message
