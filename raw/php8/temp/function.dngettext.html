dngettext

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

dngettext — Plural version of dgettext



说明

dngettext(
    string $domain,
    string $singular,
    string $plural,
    int $count
): string

The dngettext() function allows you to override the current domain for a single plural message lookup.



参数

domain

The domain

singular

plural

count


返回值

A string on success.
