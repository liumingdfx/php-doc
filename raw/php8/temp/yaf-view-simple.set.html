Yaf_View_Simple::__set

(Yaf >=1.0.0)

Yaf_View_Simple::__set — 为视图引擎分配一个模板变量



说明

public Yaf_View_Simple::__set(string $name, mixed $value): void

这是一个更简单并且用来替代 {@link Yaf_View_Simple::assign()} 的方法



参数

name
一个字符串值的名字
value

Mixed value.



返回值



示例

示例 #1 Yaf_View_Simple::__set() 示例

<?php
class IndexController extends Yaf_Controller_Abstract {
public function indexAction() {
$this->getView()->foo = "bar"; // same as assign("foo", "bar");
}
}
?>


参见

  • {@link Yaf_View_Simple::assignRef()} - The assignRef purpose
  • {@link Yaf_View_Interface::assign()} - 为视图引擎分配模板变量
