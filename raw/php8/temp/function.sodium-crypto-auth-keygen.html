sodium_crypto_auth_keygen

(PHP 7 >= 7.2.0, PHP 8)

sodium_crypto_auth_keygen — Generate a random key for sodium_crypto_auth



说明

sodium_crypto_auth_keygen(): string

Generate a key for use with {@link sodium_crypto_auth()} and {@link sodium_crypto_auth_verify()}.



参数

此函数没有参数。


返回值

Returns a 256-bit random key.
