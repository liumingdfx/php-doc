PDO_ODBC ODBC v3（IBM DB2、unixODBC 和 win32 ODBC）
PDO_PGSQL PostgreSQL
PDO_SQLITE SQLite 3 及 SQLite 2
PDO_SQLSRV Microsoft SQL Server / SQL Azure
