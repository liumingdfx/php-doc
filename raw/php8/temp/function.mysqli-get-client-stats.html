mysqli_get_client_stats

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

mysqli_get_client_stats — 返回客户端进程统计信息



说明

mysqli_get_client_stats(): array

返回客户端每个进程的统计信息。

注意:

仅可用于 mysqlnd。



参数



返回值

返回包含客户端状态的数组。



示例

示例 #1 A mysqli_get_client_stats() 示例

<?php
$link = mysqli_connect();
print_r(mysqli_get_client_stats());
?>
