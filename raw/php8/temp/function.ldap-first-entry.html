ldap_first_entry

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_first_entry — Return first result id



说明

ldap_first_entry(LDAP\Connection $ldap, LDAP\Result $result): LDAP\ResultEntry|false

Returns the entry identifier for first entry in the result. This entry identifier is then supplied to {@link ldap_next_entry()} routine to get successive entries from the result.

Entries in the LDAP result are read sequentially using the ldap_first_entry() and {@link ldap_next_entry()} functions.



参数

ldap

通过 {@link ldap_connect()} 返回的 LDAP\Connection 实例。

result

通过 {@link ldap_list()} 或者 {@link ldap_search()} 返回的 LDAP\Result 实例。



返回值

Returns an LDAP\ResultEntry instance, 或者在失败时返回 false.



更新日志

版本 说明
