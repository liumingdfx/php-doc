DOMElement::removeAttribute

(PHP 5, PHP 7, PHP 8)

DOMElement::removeAttribute — Removes attribute



说明

public DOMElement::removeAttribute(string $qualifiedName): bool

Removes attribute named qualifiedName from the element.



参数

qualifiedName

The name of the attribute.



返回值

成功时返回 true， 或者在失败时返回 false。



错误／异常

DOM_NO_MODIFICATION_ALLOWED_ERR

Raised if the node is readonly.



参见

  • {@link DOMElement::hasAttribute()} - Checks to see if attribute exists
  • {@link DOMElement::getAttribute()} - Returns value of attribute
  • {@link DOMElement::setAttribute()} - Adds new or modifies existing attribute
