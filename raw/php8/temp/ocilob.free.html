OCILob::free

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCILob::free — Frees resources associated with the LOB descriptor



说明

public OCILob::free(): bool

Frees resources associated with the descriptor, previously allocated with {@link oci_new_descriptor()}.



参数

此函数没有参数。


返回值

成功时返回 true， 或者在失败时返回 false。



更新日志

版本 说明
8.0.0, PECL OCI8 3.0.0 The OCI-Lob class was renamed to OCILob to align with PHP naming standards.
