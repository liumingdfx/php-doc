The DOMAttr class

(PHP 5, PHP 7, PHP 8)

简介

DOMAttr represents an attribute in the DOMElement object.

属性

name

The name of the attribute.

ownerElement

The element which contains the attribute or null.

schemaTypeInfo

Not implemented yet, always is null.

specified

Not implemented yet, always is true.

value

The value of the attribute.

注意:

Note, XML entities are expanded upon setting a value. Thus the & character has a special meaning. Setting value to itself will fail when value contains an &. To avoid entity expansion, use {@link DOMElement::setAttribute()} instead.
