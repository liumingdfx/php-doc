mb_list_encodings

(PHP 5, PHP 7, PHP 8)

mb_list_encodings — 返回所有支持编码的数组



说明

mb_list_encodings(): array

返回所有支持编码的数组。



参数

此函数没有参数。


返回值

返回一个数字索引数组。



错误／异常

该函数不会触发任何错误。



示例

示例 #1 mb_list_encodings() 例子

<?php

print_r(mb_list_encodings());

?>
