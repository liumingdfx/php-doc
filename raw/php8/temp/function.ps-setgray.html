ps_setgray

(PECL ps >= 1.1.0)

ps_setgray — Sets gray value



说明

ps_setgray(resource $psdoc, float $gray): bool

Sets the gray value for all following drawing operations.



参数

psdoc

Resource identifier of the postscript file as returned by {@link ps_new()}.

gray

The value must be between 0 (white) and 1 (black).



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link ps_setcolor()} - Sets current color
