DOMElement::removeAttributeNS

(PHP 5, PHP 7, PHP 8)

DOMElement::removeAttributeNS — Removes attribute



说明

public DOMElement::removeAttributeNS(?string $namespace, string $localName): void

Removes attribute localName in namespace namespace from the element.



参数

namespace

The namespace URI.

localName

The local name.



返回值

没有返回值。



错误／异常

DOM_NO_MODIFICATION_ALLOWED_ERR

Raised if the node is readonly.



参见

  • {@link DOMElement::hasAttributeNS()} - Checks to see if attribute exists
  • {@link DOMElement::getAttributeNS()} - Returns value of attribute
  • {@link DOMElement::setAttributeNS()} - Adds new attribute
