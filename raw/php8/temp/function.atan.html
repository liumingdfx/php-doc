atan

(PHP 4, PHP 5, PHP 7, PHP 8)

atan — 反正切



说明

atan(float $num): float

返回 num 的反正切值，单位是弧度。atan() 是 {@link tan()} 的反函数，它的意思是对于 atan() 的定义域中的每个 num 值，$num == tan(atan($num))。



参数

num

要处理的参数



返回值

num 的反正切弧度。



参见

  • {@link tan()} - 正切
  • {@link atanh()} - 反双曲正切
  • {@link asin()} - 反正弦
  • {@link acos()} - 反余弦
