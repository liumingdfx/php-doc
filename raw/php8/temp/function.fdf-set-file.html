fdf_set_file

(PHP 4, PHP 5 < 5.3.0, PECL fdf SVN)

fdf_set_file — Set PDF document to display FDF data in



说明

fdf_set_file(resource $fdf_document, string $url, string $target_frame = ?): bool

Selects a different PDF document to display the form results in then the form it originated from.



参数

fdf_document

The FDF document handle, returned by {@link fdf_create()}, {@link fdf_open()} or {@link fdf_open_string()}.

url

Should be given as an absolute URL.

target_frame

Use this parameter to specify the frame in which the document will be displayed. You can also set the default value for this parameter using {@link fdf_set_target_frame()}.



返回值

成功时返回 true， 或者在失败时返回 false。



示例

示例 #1 Passing FDF data to a second form

<?php
// set content type for Adobe FDF
fdf_header();

// start new fdf
$fdf = fdf_create();

// set field "foo" to value "bar"
fdf_set_value($fdf, "foo", "bar");

// tell client to display FDF data using "fdf_form.pdf"
fdf_set_file($fdf, "http://www.example.com/fdf_form.pdf");

// output fdf
fdf_save($fdf);

// clean up
fdf_close($fdf);
?>
