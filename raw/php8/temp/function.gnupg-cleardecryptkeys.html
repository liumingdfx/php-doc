gnupg_cleardecryptkeys

(PECL gnupg >= 0.5)

gnupg_cleardecryptkeys — Removes all keys which were set for decryption before



说明

gnupg_cleardecryptkeys(resource $identifier): bool


参数

identifier

gnupg 标识符，由对{@link gnupg_init()} 或 gnupg 的调用生成。


返回值

成功时返回 true， 或者在失败时返回 false。



示例

示例 #1 Procedural gnupg_cleardecryptkeys() example

<?php
$res = gnupg_init();
gnupg_cleardecryptkeys($res);
?>

示例 #2 OO gnupg_cleardecryptkeys() example

<?php
$gpg = new gnupg();
$gpg->cleardecryptkeys();
?>
