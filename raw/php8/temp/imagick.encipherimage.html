Imagick::encipherImage

(PECL imagick 2 >= 2.3.0, PECL imagick 3)

Imagick::encipherImage — Enciphers an image



说明

public Imagick::encipherImage(string $passphrase): bool

Converts plain pixels to enciphered pixels. The image is not readable until it has been deciphered using {@link Imagick::decipherImage()} 此方法在Imagick基于ImageMagick 6.3.9以上版本编译时可用。



参数

passphrase

The passphrase



返回值

成功时返回 true。



参见

  • {@link Imagick::decipherImage()} - Deciphers an image
