DOMNode::normalize

(PHP 5, PHP 7, PHP 8)

DOMNode::normalize — Normalizes the node



说明

public DOMNode::normalize(): void

Remove empty text nodes and merge adjacent text nodes in this node and all its children.



参数

此函数没有参数。


返回值

没有返回值。
