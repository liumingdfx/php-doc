The DOMNamedNodeMap class

(PHP 5, PHP 7, PHP 8)

类摘要

class DOMNamedNodeMap implements IteratorAggregate, Countable {
// 属性
public readonly int $length;
// 方法
public count(): int
public getIterator(): Iterator
public getNamedItem(string $qualifiedName): ?DOMNode
public getNamedItemNS(?string $namespace, string $localName): ?DOMNode
public item(int $index): ?DOMNode
}

属性

length

The number of nodes in the map. The range of valid child node indices is 0 to length - 1 inclusive.

更新日志

版本 说明

目录

  • {@link DOMNamedNodeMap::count} — Get number of nodes in the map
  • {@link DOMNamedNodeMap::getIterator} — Retrieve an external iterator
  • {@link DOMNamedNodeMap::getNamedItem} — Retrieves a node specified by name
  • {@link DOMNamedNodeMap::getNamedItemNS} — Retrieves a node specified by local name and namespace URI
  • {@link DOMNamedNodeMap::item} — Retrieves a node specified by index
