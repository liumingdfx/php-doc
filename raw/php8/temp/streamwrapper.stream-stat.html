streamWrapper::stream_stat

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

streamWrapper::stream_stat — Retrieve information about a file resource



说明

public streamWrapper::stream_stat(): array|false

This method is called in response to {@link fstat()}.



参数

此函数没有参数。


返回值

See {@link stat()}.



错误／异常

调用此方法失败将给出E_WARNING（未实现）。



参见

  • {@link stat()} - 给出文件的信息
  • {@link streamwrapper::url_stat()} - Retrieve information about a file
