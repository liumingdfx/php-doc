sodium_crypto_aead_chacha20poly1305_ietf_keygen

(PHP 7 >= 7.2.0, PHP 8)

sodium_crypto_aead_chacha20poly1305_ietf_keygen — Generate a random ChaCha20-Poly1305 (IETF) key.



说明

sodium_crypto_aead_chacha20poly1305_ietf_keygen(): string

Generate a random key for use with {@link sodium_crypto_aead_chacha20poly1305_ietf_encrypt()} and {@link sodium_crypto_aead_chacha20poly1305_ietf_decrypt()}.



参数

此函数没有参数。


返回值

Returns a 256-bit random key.
