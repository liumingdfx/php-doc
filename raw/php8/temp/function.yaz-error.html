yaz_error

(PHP 4 >= 4.0.1, PECL yaz >= 0.9.0)

yaz_error — Returns error description



说明

yaz_error(resource $id): string

yaz_error() returns an English text message corresponding to the last error number as returned by {@link yaz_errno()}.



参数

id

The connection resource returned by {@link yaz_connect()}.



返回值

Returns an error text message for server (last request), identified by parameter id. An empty string is returned if the last operation was successful.



参见

  • {@link yaz_errno()} - Returns error number
  • {@link yaz_addinfo()} - Returns additional error information
