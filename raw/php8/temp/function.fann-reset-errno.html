fann_reset_errno

(PECL fann >= 1.0.0)

fann_reset_errno — 重置最后的错误代码。



说明

fann_reset_errno(resource $errdat): void

重置最后的错误代码。



参数

errdat
Either neural network resource or neural network trainining data resource.


返回值

无返回值。



参见

  • {@link fann_get_errno()} - 返回最后一个错误数字。
  • {@link fann_reset_errstr()} - 重置最后的错误字符串。
