Gmagick::setCompressionQuality

(No version information available, might only be in Git)

Gmagick::setCompressionQuality — Sets the object's default compression quality



说明

Gmagick::setCompressionQuality( int $quality = 75 ): Gmagick

Sets the object's default compression quality.



参数

quality


返回值

The Gmagick object.



错误／异常

错误时抛出GmagickException。



示例

示例 #1 Gmagick::setCompressionQuality()

<?php
$gm = new Gmagick();
$gm->read("magick:rose");
$gm->setCompressionQuality(2);
?>
