Imagick::getImageGravity

(PECL imagick 2 >= 2.3.0, PECL imagick 3)

Imagick::getImageGravity — Gets the image gravity



说明

public Imagick::getImageGravity(): int

Gets the current gravity value of the image. Unlike {@link Imagick::getGravity()}, this method returns the gravity defined for the current image sequence. 此方法在Imagick基于ImageMagick 6.4.4以上版本编译时可用。



参数

此函数没有参数。


返回值

Returns the images gravity property. Refer to the list of gravity constants.
