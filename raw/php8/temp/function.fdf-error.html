fdf_error

(PHP 4 >= 4.3.0, PHP 5 < 5.3.0, PECL fdf SVN)

fdf_error — Return error description for FDF error code



说明

fdf_error(int $error_code = -1): string

Gets a textual description for the FDF error code given in error_code.



参数

error_code

An error code obtained with {@link fdf_errno()}. If not provided, this function uses the internal error code set by the last operation.



返回值

Returns the error message as a string, or the string no error if nothing went wrong.



参见

  • {@link fdf_errno()} - Return error code for last fdf operation
