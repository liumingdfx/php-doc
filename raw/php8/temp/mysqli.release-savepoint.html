mysqli::release_savepoint

mysqli_release_savepoint

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

mysqli::release_savepoint -- mysqli_release_savepoint — 从当前事务的保存点中移除一个命名保存点



说明

面向对象风格

public mysqli::release_savepoint(string $name): bool

过程化风格:

mysqli_release_savepoint(mysqli $mysql, string $name): bool

此函数等同于执行 $mysqli->query("RELEASE SAVEPOINT `$name`");。此函数不会触发 commit 或者 rollback。



参数

mysql

仅以过程化样式：由 {@link mysqli_connect()} 或mysqli_init() 返回的 mysqli 对象。

name

保存点的标识符。



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • mysqli_savepoint() - 在当前事务中设置命名保存点
