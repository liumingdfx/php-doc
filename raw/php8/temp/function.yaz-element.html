yaz_element

(PHP 4 >= 4.0.1, PECL yaz >= 0.9.0)

yaz_element — Specifies Element-Set Name for retrieval



说明

yaz_element(resource $id, string $elementset): bool

This function sets the element set name for retrieval.

Call this function before {@link yaz_search()} or {@link yaz_present()} to specify the element set name for records to be retrieved.

注意:

If this function appears to have no effect, see the description of the piggybacking option in {@link yaz_connect()}.



参数

id

The connection resource returned by {@link yaz_connect()}.

elementset

Most servers support F (for full records) and B (for brief records).



返回值

成功时返回 true， 或者在失败时返回 false。
