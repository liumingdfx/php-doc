fann_set_weight

(PECL fann >= 1.0.0)

fann_set_weight — 在网络中设置一个连接。



说明

fann_set_weight(
    resource $ann,
    int $from_neuron,
    int $to_neuron,
    float $weight
): bool

在网络中设置一个连接。



参数

ann
神经网络 资源。
from_neuron

连接开始处的神经元。

to_neuron

连接结束处的神经元。

weight

连接权重。



返回值

成功时返回 true，其它情况下返回 false。
