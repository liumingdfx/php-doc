Collection::getSession — Get session object



说明

public mysql_xdevapi\Collection::getSession(): Session

Get a new Session object from the Collection object.



参数

此函数没有参数。


返回值

A Session object.



示例

示例 #1 mysql_xdevapi\Collection::getSession() example

<?php
$session = mysql_xdevapi\getSession("mysqlx://user:password@localhost");
$session->sql("DROP DATABASE IF EXISTS addressbook")->execute();
$session->sql("CREATE DATABASE addressbook")->execute();

$schema = $session->getSchema("addressbook");
$collection = $schema->createCollection("people");

// ...

$newsession = $collection->getSession();

var_dump($session);
var_dump($newsession);
?>

以上示例的输出类似于：

}
