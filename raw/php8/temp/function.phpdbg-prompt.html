phpdbg_prompt

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

phpdbg_prompt — Sets the command prompt



说明

phpdbg_prompt(string $string): void

Set the command prompt to the given string.



参数

string
The string to use as command prompt.


返回值

没有返回值。
