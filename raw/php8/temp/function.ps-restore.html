ps_restore

(PECL ps >= 1.1.0)

ps_restore — Restore previously save context



说明

ps_restore(resource $psdoc): bool

Restores a previously saved graphics context. Any call of {@link ps_save()} must be accompanied by a call to ps_restore(). All coordinate transformations, line style settings, color settings, etc. are being restored to the state before the call of {@link ps_save()}.



参数

psdoc

Resource identifier of the postscript file as returned by {@link ps_new()}.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link ps_save()} - Save current context
