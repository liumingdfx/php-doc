ps_set_border_color

(PECL ps >= 1.1.0)

ps_set_border_color — Sets color of border for annotations



说明

ps_set_border_color(
    resource $psdoc,
    float $red,
    float $green,
    float $blue
): bool

Links added with one of the functions {@link ps_add_weblink()}, {@link ps_add_pdflink()}, etc. will be displayed with a surounded rectangle when the postscript document is converted to pdf and viewed in a pdf viewer. This rectangle is not visible in the postscript document. This function sets the color of the rectangle's border line.



参数

psdoc

Resource identifier of the postscript file as returned by {@link ps_new()}.

red

The red component of the border color.

green

The green component of the border color.

blue

The blue component of the border color.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link ps_set_border_dash()} - Sets length of dashes for border of annotations
  • {@link ps_set_border_style()} - Sets border style of annotations
