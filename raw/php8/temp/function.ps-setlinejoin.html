ps_setlinejoin

(PECL ps >= 1.1.0)

ps_setlinejoin — Sets how contected lines are joined



说明

ps_setlinejoin(resource $psdoc, int $type): bool

Sets how lines are joined.



参数

psdoc

Resource identifier of the postscript file as returned by {@link ps_new()}.

type

The way lines are joined. Possible values are PS_LINEJOIN_MITER, PS_LINEJOIN_ROUND, or PS_LINEJOIN_BEVEL.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link ps_setlinecap()} - Sets appearance of line ends
  • {@link ps_setlinewidth()} - Sets width of a line
  • {@link ps_setmiterlimit()} - Sets the miter limit
