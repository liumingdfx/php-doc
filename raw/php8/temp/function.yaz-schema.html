yaz_schema

(PHP 4 >= 4.2.0, PECL yaz >= 0.9.0)

yaz_schema — Specifies schema for retrieval



说明

yaz_schema(resource $id, string $schema): void

yaz_schema() specifies the schema for retrieval.

This function should be called before {@link yaz_search()} or {@link yaz_present()}.



参数

id

The connection resource returned by {@link yaz_connect()}.

schema

Must be specified as an OID (Object Identifier) in a raw dot-notation (like 1.2.840.10003.13.4) or as one of the known registered schemas: GILS-schema, Holdings, Zthes, ...



返回值

没有返回值。
