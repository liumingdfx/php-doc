OCILob::close

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCILob::close — Closes LOB descriptor



说明

public OCILob::close(): bool

Closes descriptor of LOB or FILE. This function should be used only with {@link OCILob::writeTemporary}.



参数

此函数没有参数。


返回值

成功时返回 true， 或者在失败时返回 false。

8.0.0, PECL OCI8 3.0.0 The OCI-Lob class was renamed to OCILob to align with PHP naming standards.
