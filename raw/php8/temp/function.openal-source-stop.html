openal_source_stop

(PECL openal >= 0.1.0)

openal_source_stop — Stop playing the source



说明

openal_source_stop(resource $source): bool


参数

source
An Open AL(Source) resource (previously created by {@link openal_source_create()}).


返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link openal_source_play()} - Start playing the source
  • {@link openal_source_pause()} - Pause the source
  • {@link openal_source_rewind()} - Rewind the source
