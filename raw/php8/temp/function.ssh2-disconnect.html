ssh2_disconnect

(PECL ssh2 >= 1.0)

ssh2_disconnect — Close a connection to a remote SSH server



说明

ssh2_disconnect(resource $session): bool

Close a connection to a remote SSH server.



参数

session
An SSH connection link identifier, obtained from a call to {@link ssh2_connect()}.


返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link ssh2_connect()} - Connect to an SSH server
