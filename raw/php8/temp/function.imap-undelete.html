imap_undelete

(PHP 4, PHP 5, PHP 7, PHP 8)

imap_undelete — Unmark the message which is marked deleted



说明

imap_undelete(IMAP\Connection $imap, string $message_nums, int $flags = 0): true

Removes the deletion flag for a specified message, which is set by {@link imap_delete()} or {@link imap_mail_move()}.



参数

imap

IMAP\Connection 实例。

message_nums

A string representing one or more messages in IMAP4-style sequence format ("n", "n:m", or combination of these delimited by commas).

flags


返回值

总是返回 true。



更新日志

版本 说明
8.1.0 现在 imap 参数接受 IMAP\Connection 实例，之前接受有效的 imap resource。


参见

  • {@link imap_delete()} - Mark a message for deletion from current mailbox
  • {@link imap_mail_move()} - Move specified messages to a mailbox
