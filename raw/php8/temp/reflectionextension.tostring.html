ReflectionExtension::__toString

(PHP 5, PHP 7, PHP 8)

ReflectionExtension::__toString — 生成字符串



说明

public ReflectionExtension::__toString(): string

以 string 形式返回扩展的反射信息。等同于 {@link ReflectionExtension::export()} return 参数设置为 true。



参数

此函数没有参数。


返回值

返回扩展的反射信息，同 {@link ReflectionExtension::export()}。



参见

  • {@link ReflectionExtension::__construct()} - 构造 ReflectionExtension
  • {@link ReflectionExtension::export()} - Export
  • __toString()
