MongoDB\Driver\Cursor::__construct

(mongodb >=1.0.0)

MongoDB\Driver\Cursor::__construct — Create a new Cursor (not used)



说明

final private MongoDB\Driver\Cursor::__construct()

MongoDB\Driver\Cursor objects are returned as the result of an executed command or query and cannot be constructed directly.



参数

此函数没有参数。


参见

  • {@link MongoDB\Driver\Manager::executeCommand()} - Execute a database command
  • {@link MongoDB\Driver\Manager::executeQuery()} - Execute a database query
  • {@link MongoDB\Driver\Server::executeCommand()} - Execute a database command on this server
  • {@link MongoDB\Driver\Server::executeQuery()} - Execute a database query on this server
