fdf_set_flags

(PHP 4 >= 4.0.2, PHP 5 < 5.3.0, PECL fdf SVN)

fdf_set_flags — Sets a flag of a field



说明

fdf_set_flags(
    resource $fdf_document,
    string $fieldname,
    int $whichFlags,
    int $newFlags
): bool

Sets certain flags of the given field.



参数

fdf_document

The FDF document handle, returned by {@link fdf_create()}, {@link fdf_open()} or {@link fdf_open_string()}.

fieldname

Name of the FDF field, as a string.

whichFlags

newFlags


返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link fdf_set_opt()} - Sets an option of a field
