imap_unsubscribe

(PHP 4, PHP 5, PHP 7, PHP 8)

imap_unsubscribe — Unsubscribe from a mailbox



说明

imap_unsubscribe(IMAP\Connection $imap, string $mailbox): bool

Unsubscribe from the specified mailbox.



参数

imap

IMAP\Connection 实例。

mailbox

The mailbox name, see {@link imap_open()} for more information

警告

Passing untrusted data to this parameter is insecure, unlessimap.enable_insecure_rsh is disabled.


返回值

成功时返回 true， 或者在失败时返回 false。



更新日志

版本 说明
8.1.0 现在 imap 参数接受 IMAP\Connection 实例，之前接受有效的 imap resource。
