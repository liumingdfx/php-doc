Memcache::close

(PECL memcache >= 0.4.0)

Memcache::close — 关闭memcache连接



说明

Memcache::close(): bool

Memcache::close()关闭到memcached服务端的连接。这个函数不会关闭持久化连接， 持久化连接仅仅会在web服务器关机/重启时关闭。与之对应的，你也可以使用memcache_close()函数。



参数

此函数没有参数。


返回值

成功时返回 true， 或者在失败时返回 false。



示例

示例 #1 Memcache::close()示例

<?php

// 面向过程 API
$memcache_obj = memcache_connect('memcache_host', 11211);
//
do something here...

memcache_close($memcache_obj);

// 面向对象 API
$memcache_obj = new Memcache;
$memcache_obj->connect('memcache_host', 11211);
//
do something here...

$memcache_obj->close();

?>


参见

  • {@link Memcache::connect()} - 打开一个memcached服务端连接
  • {@link Memcache::pconnect()} - 打开一个到服务器的持久化连接
