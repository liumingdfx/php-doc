Threaded::merge

(PECL pthreads >= 2.0.0)

Threaded::merge — 操作



说明

publicThreaded::merge(mixed $from, bool $overwrite = ?): bool

将数据合并到当前对象



参数

from
要合并的数据
overwrite

如果现有对象已经存在同键的数据，是否覆盖。默认为 true



返回值

成功时返回 true， 或者在失败时返回 false。



示例

示例 #1 合并数据到对象的属性表

<?php
$array = [];

while (count($array) < 10)
$array[] = count($array);

$stdClass = new stdClass();
$stdClass->foo = "foo";
$stdClass->bar = "bar";
$stdClass->baz = "baz";

$safe = new Threaded();
$safe->merge($array);
$safe->merge($stdClass);

var_dump($safe);
?>
