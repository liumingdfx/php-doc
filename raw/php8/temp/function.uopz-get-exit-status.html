uopz_get_exit_status

(PECL uopz 5, PECL uopz , PECL uopz 7)

uopz_get_exit_status — Retrieve the last set exit status



说明

uopz_get_exit_status(): mixed

Retrieves the last set exit status, i.e. the value passed to {@link exit()}.



参数

此函数没有参数。


返回值

This function returns the last exit status, or null if {@link exit()} has not been called.



示例

示例 #1 uopz_get_exit_status() example

<?php
exit(123);
echo uopz_get_exit_status();?>

以上示例会输出：

123


注释

警告

OPcache optimizes away dead code after unconditional exit.



参见

  • {@link uopz_allow_exit()} - Allows control over disabled exit opcode
