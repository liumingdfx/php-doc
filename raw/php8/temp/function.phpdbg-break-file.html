phpdbg_break_file

(PHP 5 >= 5.6.3, PHP 7, PHP 8)

phpdbg_break_file — Inserts a breakpoint at a line in a file



说明

phpdbg_break_file(string $file, int $line): void

Insert a breakpoint at the given line in the given file.



参数

file
The name of the file.
line

The line number.



返回值

没有返回值。



参见

  • {@link phpdbg_break_function()} - Inserts a breakpoint at entry to a function
  • {@link phpdbg_break_method()} - Inserts a breakpoint at entry to a method
  • {@link phpdbg_break_next()} - Inserts a breakpoint at the next opcode
  • {@link phpdbg_clear()} - Clears all breakpoints
