enchant_broker_free_dict

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL enchant >= 0.1.0 )

enchant_broker_free_dict — Free a dictionary resource

警告

本函数已自 PHP 8.0.0 起被废弃。强烈建议不要依赖本函数。



说明

enchant_broker_free_dict(EnchantDictionary $dictionary): bool

Free a dictionary. As of PHP 8.0.0, it is recommended to unset the object instead of calling this function.



参数

dictionary

An Enchant dictionary returned by {@link enchant_broker_request_dict()} or {@link enchant_broker_request_pwl_dict()}.



返回值

成功时返回 true， 或者在失败时返回 false。



更新日志

版本 说明
8.0.0 dictionary expects a EnchantDictionary now; previoulsy, a resource was expected.


参见

  • {@link enchant_broker_request_dict()} - Create a new dictionary using a tag
  • {@link enchant_broker_request_pwl_dict()} - Creates a dictionary using a PWL file
