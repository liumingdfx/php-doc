atan2

(PHP 4, PHP 5, PHP 7, PHP 8)

atan2 — 两个参数的反正切



说明

atan2(float $y, float $x): float

本函数计算两个变量 x 和 y 的反正切值。和计算 y / x 的反正切相似，只除了两个参数的符号是用来确定结果的象限之外。

本函数的结果为弧度，其值在 -PI 和 PI 之间（包括 -PI 和 PI）。



参数

y

Dividend parameter

x

Divisor parameter



返回值

x 和 y 的反正切弧度值。
