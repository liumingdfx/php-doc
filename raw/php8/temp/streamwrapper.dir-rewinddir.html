streamWrapper::dir_rewinddir

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

streamWrapper::dir_rewinddir — Rewind directory handle



说明

public streamWrapper::dir_rewinddir(): bool

This method is called in response to {@link rewinddir()}.

Should reset the output generated by {@link streamWrapper::dir_readdir()}. i.e.: The next call to {@link streamWrapper::dir_readdir()} should return the first entry in the location returned by {@link streamWrapper::dir_opendir()}.



参数

此函数没有参数。


返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link rewinddir()} - 倒回目录句柄
  • {@link streamWrapper::dir_readdir()} - Read entry from directory handle
