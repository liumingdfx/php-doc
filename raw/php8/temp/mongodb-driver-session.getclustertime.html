MongoDB\Driver\Session::getClusterTime

(mongodb >=1.4.0)

MongoDB\Driver\Session::getClusterTime — Returns the cluster time for this session



说明

final public MongoDB\Driver\Session::getClusterTime(): ?object

Returns the cluster time for this session. If the session has not been used for any operation and {@link MongoDB\Driver\Session::advanceClusterTime()} has not been called, the cluster time will be null.



参数

此函数没有参数。


返回值

Returns the cluster time for this session, or null if the session has no cluster time.



错误／异常

  • Throws MongoDB\Driver\Exception\InvalidArgumentException on argument parsing errors.


参见

  • {@link MongoDB\Driver\Session::advanceClusterTime()} - Advances the cluster time for this session
