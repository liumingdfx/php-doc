sodium_crypto_auth_verify

(PHP 7 >= 7.2.0, PHP 8)

sodium_crypto_auth_verify — Verifies that the tag is valid for the message



说明

sodium_crypto_auth_verify(string $mac, string $message, string $key): bool

Verify the authentication tag is valid for a given message and key.

Unlike with digital signatures (e.g. {@link sodium_crypto_sign_verify_detached()}), any party capable of verifying a message is also capable of authenticating their own messages. (Hence, symmetric authentication.)



参数

mac
Authentication tag produced by {@link sodium_crypto_auth()}
message

Message

key

Authentication key



返回值

成功时返回 true， 或者在失败时返回 false。
