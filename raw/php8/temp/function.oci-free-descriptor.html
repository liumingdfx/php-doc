oci_free_descriptor

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

oci_free_descriptor — Frees a descriptor



说明

oci_free_descriptor(OCILob $lob): bool

Frees a descriptor allocated by {@link oci_new_descriptor()}.



参数

descriptor

Descriptor allocated by {@link oci_new_descriptor()}.



返回值

成功时返回 true， 或者在失败时返回 false。



注释

注意:

This function is commonly used as a method {@link OCILOB::free}.



参见

  • {@link OCILOB::free}
