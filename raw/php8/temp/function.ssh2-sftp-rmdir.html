ssh2_sftp_rmdir

(PECL ssh2 >= 0.9.0)

ssh2_sftp_rmdir — Remove a directory



说明

ssh2_sftp_rmdir(resource $sftp, string $dirname): bool

Removes a directory from the remote file server.

This function is similar to using {@link rmdir()} with the ssh2.sftp:// wrapper.



参数

sftp

An SSH2 SFTP resource opened by {@link ssh2_sftp()}.

dirname


返回值

成功时返回 true， 或者在失败时返回 false。



示例

示例 #1 Removing a directory on a remote server

<?php
$connection = ssh2_connect('shell.example.com', 22);
ssh2_auth_password($connection, 'username', 'password');
$sftp = ssh2_sftp($connection);

ssh2_sftp_rmdir($sftp, '/home/username/deltodel');
// Or: rmdir("ssh2.sftp://$sftp/home/username/dirtodel");
?>


参见

  • {@link rmdir()} - 删除目录
  • {@link ssh2_sftp_mkdir()} - Create a directory
