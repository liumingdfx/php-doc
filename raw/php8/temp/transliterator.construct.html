Transliterator::__construct

(PHP 5 >= 5.4.0, PHP 7, PHP 8, PECL intl >= 2.0.0)

Transliterator::__construct — Private constructor to deny instantiation



说明

final private Transliterator::__construct()

This method should not be called. Its only purpose is to deny instantiation with the new operator.

Use the factory methods {@link Transliterator::create()} or {@link Transliterator::createFromRules()} instead.



参数

此函数没有参数。


参见

  • {@link Transliterator::create()} - Create a transliterator
  • {@link Transliterator::createFromRules()} - Create transliterator from rules
