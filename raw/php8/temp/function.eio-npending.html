eio_npending

(PECL eio >= 0.0.1dev)

eio_npending — Returns number of finished, but unhandled requests



说明

eio_npending(): int

eio_npending() returns number of finished, but unhandled requests



参数

此函数没有参数。


返回值

eio_npending() returns number of finished, but unhandled requests.



参见

  • {@link eio_nreqs()} - Returns number of requests to be processed
  • {@link eio_nready()} - Returns number of not-yet handled requests
  • {@link eio_nthreads()} - Returns number of threads currently in use
