db2_result

(PECL ibm_db2 >= 1.0.0)

db2_result — Returns a single column from a row in the result set



说明

db2_result(resource $stmt, int|string $column): ?mixed

Use db2_result() to return the value of a specified column in the current row of a result set. You must call {@link db2_fetch_row()} before calling db2_result() to set the location of the result set pointer.



参数

stmt

A valid stmt resource.

column

Either an integer mapping to the 0-indexed field in the result set, or a string matching the name of the column.



返回值

Returns the value of the requested field if the field exists in the result set. Returns null if the field does not exist, and issues a warning.



示例

示例 #1 A db2_result() example

The following example demonstrates how to iterate through a result set with {@link db2_fetch_row()} and retrieve columns from the result set with db2_result().

<?php
$sql = 'SELECT name, breed FROM animals WHERE weight < ?';
$stmt = db2_prepare($conn, $sql);
db2_execute($stmt, array(10));
while (db2_fetch_row($stmt)) {
$name = db2_result($stmt, 0);
$breed = db2_result($stmt, 'BREED');
print "$name $breed";
}
?>

以上示例会输出：

cat Pook
gold fish Bubbles
budgerigar Gizmo
goat Rickety Ride


参见

  • {@link db2_fetch_array()} - Returns an array, indexed by column position, representing a row in a result set
  • {@link db2_fetch_assoc()} - Returns an array, indexed by column name, representing a row in a result set
  • {@link db2_fetch_both()} - Returns an array, indexed by both column name and position, representing a row in a result set
  • {@link db2_fetch_object()} - Returns an object with properties representing columns in the fetched row
  • {@link db2_fetch_row()} - Sets the result set pointer to the next row or requested row
