mysqli::dump_debug_info

mysqli_dump_debug_info

(PHP 5, PHP 7, PHP 8)

mysqli::dump_debug_info -- mysqli_dump_debug_info — 将调试信息输出到日志



说明

面向对象风格

public mysqli::dump_debug_info(): bool

过程化风格

mysqli_dump_debug_info(mysqli $mysql): bool

这个函数设计用于超级权限用户执行将调试信息输出到连接相关的 MySQL Server 的日志中。



参数

mysql

仅以过程化样式：由 {@link mysqli_connect()} 或mysqli_init() 返回的 mysqli 对象。


返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • mysqli_debug() - Performs debugging operations
