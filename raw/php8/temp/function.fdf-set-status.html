fdf_set_status

(PHP 4, PHP 5 < 5.3.0, PECL fdf SVN)

fdf_set_status — Set the value of the /STATUS key



说明

fdf_set_status(resource $fdf_document, string $status): bool

Sets the value of the /STATUS key. When a client receives a FDF with a status set it will present the value in an alert box.



参数

fdf_document

The FDF document handle, returned by {@link fdf_create()}, {@link fdf_open()} or {@link fdf_open_string()}.

status


返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link fdf_get_status()} - Get the value of the /STATUS key
