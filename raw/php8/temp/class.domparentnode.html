The DOMParentNode interface

(PHP 8)

接口摘要

interface DOMParentNode {
// 方法
public append(DOMNode|string ...$nodes): void
public prepend(DOMNode|string ...$nodes): void
public replaceChildren(DOMNode|string ...$nodes): void
}

目录

  • {@link DOMParentNode::append} — Appends nodes after the last child node
  • {@link DOMParentNode::prepend} — Prepends nodes before the first child node
  • {@link DOMParentNode::replaceChildren} — Replace children in node
