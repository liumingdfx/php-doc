Ds\Vector::get

(PECL ds >= 1.0.0)

Ds\Vector::get — Returns the value at a given index



说明

public Ds\Vector::get(int $index): mixed

Returns the value at a given index.



参数

index
The index to access, starting at 0.


返回值

The value at the requested index.



错误／异常

OutOfRangeException if the index is not valid.



示例

示例 #1 Ds\Vector::get() example

<?php
$vector = new \Ds\Vector(["a", "b", "c"]);

var_dump($vector->get(0));
var_dump($vector->get(1));
var_dump($vector->get(2));
?>

以上示例的输出类似于：

string(1) "a"
string(1) "b"
string(1) "c"

示例 #2 Ds\Vector::get() example using array syntax

<?php
$vector = new \Ds\Vector(["a", "b", "c"]);

var_dump($vector[0]);
var_dump($vector[1]);
var_dump($vector[2]);
?>

以上示例的输出类似于：

string(1) "a"
string(1) "b"
string(1) "c"
