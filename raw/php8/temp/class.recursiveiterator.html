The RecursiveIterator interface

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

简介

Classes implementing RecursiveIterator can be used to iterate over iterators recursively.

接口摘要

interface RecursiveIterator extends Iterator {
// 方法
public getChildren(): ?RecursiveIterator
public hasChildren(): bool
// 继承的方法
public {@link Iterator::current}(): mixed
public {@link Iterator::key}(): mixed
public {@link Iterator::next}(): void
public {@link Iterator::rewind}(): void
public {@link Iterator::valid}(): bool
}

目录

  • {@link RecursiveIterator::getChildren} — Returns an iterator for the current entry
  • {@link RecursiveIterator::hasChildren} — Returns if an iterator can be created for the current entry
