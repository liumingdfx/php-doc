odbc_fetch_array

(PHP 4 >= 4.0.2, PHP 5, PHP 7, PHP 8)

odbc_fetch_array — Fetch a result row as an associative array



说明

odbc_fetch_array(resource $statement, int $row = -1): array|false

Fetch an associative array from an ODBC query.



参数

statement

The result resource from {@link odbc_exec()}.

row

Optionally choose which row number to retrieve.



返回值

Returns an array that corresponds to the fetched row, or false if there are no more rows.



注释

注意: This function exists when compiled with DBMaker, IBM DB2 or UnixODBC support.



参见

  • {@link odbc_fetch_row()} - Fetch a row
  • {@link odbc_fetch_object()} - Fetch a result row as an object
  • {@link odbc_num_rows()} - Number of rows in a result
