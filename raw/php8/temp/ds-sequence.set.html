Ds\Sequence::set

(PECL ds >= 1.0.0)

Ds\Sequence::set — Updates a value at a given index



说明

abstract public Ds\Sequence::set(int $index, mixed $value): void

Updates a value at a given index.



参数

index
The index of the value to update.
value

The new value.



返回值

没有返回值。



错误／异常

OutOfRangeException if the index is not valid.



示例

示例 #1 Ds\Sequence::set() example

<?php
$sequence = new \Ds\Vector(["a", "b", "c"]);

$sequence->set(1, "_");
print_r($sequence);
?>

以上示例的输出类似于：

Ds\Vector Object
(
[0] => a
[1] => _
[2] => c
)

示例 #2 Ds\Sequence::set() example using array syntax

<?php
$sequence = new \Ds\Vector(["a", "b", "c"]);

$sequence[1] = "_";
print_r($sequence);
?>

以上示例的输出类似于：

Ds\Vector Object
(
[0] => a
[1] => _
[2] => c
)
