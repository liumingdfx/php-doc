decbin

(PHP 4, PHP 5, PHP 7, PHP 8)

decbin — 十进制转换为二进制



说明

decbin(int $num): string

返回字符串，包含有给定 num 参数的二进制表示。

num

示例

示例 #1 decbin() 示例

<?php
echo decbin(12) . "\n";
echo decbin(26);
?>

以上示例会输出：

1100
11010
