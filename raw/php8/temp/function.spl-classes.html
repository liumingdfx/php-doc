spl_classes

(PHP 5, PHP 7, PHP 8)

spl_classes — 返回所有可用的SPL类



说明

spl_classes(): array

本函数返回当前所有可用的 SPL 类的数组。



参数

此函数没有参数。


返回值

Returns an array containing the currently available SPL classes.



示例

示例 #1 spl_classes() example

<?php

print_r(spl_classes());

?>

以上示例的输出类似于：
