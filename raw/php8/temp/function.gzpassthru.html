gzpassthru

(PHP 4, PHP 5, PHP 7, PHP 8)

gzpassthru — Output all remaining data on a gz-file pointer



说明

gzpassthru(resource $stream): int

Reads to EOF on the given gz-file pointer from the current position and writes the (uncompressed) results to standard output.

注意:

You may need to call {@link gzrewind()} to reset the file pointer to the beginning of the file if you have already written data to it.

小技巧

If you just want to dump the contents of a file to the output buffer, without first modifying it or seeking to a particular offset, you may want to use the {@link readgzfile()} function, which saves you the {@link gzopen()} call.



参数

stream

The gz-file pointer. It must be valid, and must point to a file successfully opened by {@link gzopen()}.



返回值

The number of uncompressed characters read from gz and passed through to the input.



示例

示例 #1 gzpassthru() example

<?php
$fp = gzopen('file.gz', 'r');
gzpassthru($fp);
gzclose($fp);
?>
