mailparse_msg_create

(PECL mailparse >= 0.9.0)

mailparse_msg_create — Create a mime mail resource



说明

mailparse_msg_create(): resource

Create a MIME mail resource.



参数

此函数没有参数。


返回值

Returns a handle that can be used to parse a message.



注释

注意:

It is recommended to call {@link mailparse_msg_free()} on the result of this function, when it is no longer needed, to avoid memory leaks.



参见

  • {@link mailparse_msg_free()} - Frees a MIME resource
  • {@link mailparse_msg_parse_file()} - Parses a file
