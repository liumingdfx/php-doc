gnupg_encrypt

(PECL gnupg >= 0.1)

gnupg_encrypt — Encrypts a given text



说明

gnupg_encrypt(resource $identifier, string $plaintext): string|false

Encrypts the given plaintext with the keys, which were set with {@link gnupg_addencryptkey} before and returns the encrypted text.



参数

identifier

gnupg 标识符，由对{@link gnupg_init()} 或 gnupg 的调用生成。

plaintext

The text being encrypted.



返回值

On success, this function returns the encrypted text. On failure, this function returns false.



示例

示例 #1 Procedural gnupg_encrypt() example

<?php
$res = gnupg_init();
gnupg_addencryptkey($res,"8660281B6051D071D94B5B230549F9DC851566DC");
$enc = gnupg_encrypt($res, "just a test");
echo $enc;
?>

示例 #2 OO gnupg_encrypt() example

<?php
$gpg = new gnupg();
$gpg->addencryptkey("8660281B6051D071D94B5B230549F9DC851566DC");
$enc = $gpg->encrypt("just a test");
echo $enc;
?>
