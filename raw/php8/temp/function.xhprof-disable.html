xhprof_disable

(PECL xhprof >= 0.9.0)

xhprof_disable — 停止 xhprof 分析器



说明

xhprof_disable(): array

停止性能分析，并返回此次运行的 xhprof 数据。



参数

此函数没有参数。


返回值

本次运行的 array 类型的 xhprof 数据。



示例

示例 #1 xhprof_disable() 范例

<?php
xhprof_enable();

$foo = strlen("foo bar");

$xhprof_data = xhprof_disable();

print_r($xhprof_data);
?>

以上示例的输出类似于：

Array
(
[main()==>strlen] => Array
(
[ct] => 1
[wt] => 279
)

[main()==>xhprof_disable] => Array
(
[ct] => 1
[wt] => 9
)

[main()] => Array
(
[ct] => 1
[wt] => 610
)

)
