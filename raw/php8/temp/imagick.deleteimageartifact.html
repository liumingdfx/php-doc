Imagick::deleteImageArtifact

(PECL imagick 3)

Imagick::deleteImageArtifact — Delete image artifact



说明

public Imagick::deleteImageArtifact(string $artifact): bool

Deletes an artifact associated with the image. The difference between image properties and image artifacts is that properties are public and artifacts are private. 此方法在Imagick基于ImageMagick 6.5.7以上版本编译时可用。



参数

artifact

The name of the artifact to delete



返回值

成功时返回 true。



错误／异常

错误时抛出 ImagickException。



参见

  • {@link Imagick::setImageArtifact()} - Set image artifact
  • {@link Imagick::getImageArtifact()} - Get image artifact
