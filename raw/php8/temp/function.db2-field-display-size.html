db2_field_display_size

(PECL ibm_db2 >= 1.0.0)

db2_field_display_size — Returns the maximum number of bytes required to display a column



说明

db2_field_display_size(resource $stmt, int|string $column): int|false

Returns the maximum number of bytes required to display a column in a result set.



参数

stmt

Specifies a statement resource containing a result set.

column

Specifies the column in the result set. This can either be an integer representing the 0-indexed position of the column, or a string containing the name of the column.



返回值

Returns an integer value with the maximum number of bytes required to display the specified column. If the column does not exist in the result set, db2_field_display_size() returns false.



参见

  • {@link db2_field_name()} - Returns the name of the column in the result set
  • {@link db2_field_num()} - Returns the position of the named column in a result set
  • {@link db2_field_precision()} - Returns the precision of the indicated column in a result set
  • {@link db2_field_scale()} - Returns the scale of the indicated column in a result set
  • {@link db2_field_type()} - Returns the data type of the indicated column in a result set
  • {@link db2_field_width()} - Returns the width of the current value of the indicated column in a result set
