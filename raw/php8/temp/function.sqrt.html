sqrt

(PHP 4, PHP 5, PHP 7, PHP 8)

sqrt — 平方根



说明

sqrt(float $num): float

返回 num 的平方根。



参数

num

要处理的参数



返回值

返回 num 的平方根，负数时返回 NAN。



示例

示例 #1 sqrt() 示例

<?php
// 精度取决于精度指令
echo sqrt(9); // 3
echo sqrt(10); // 3.16227766 ...
?>
