session_status

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

session_status — 返回当前会话状态



说明

session_status(): int

session_status() 被用于返回当前会话状态。



参数

此函数没有参数。


返回值

  • PHP_SESSION_DISABLED 会话是被禁用的。
  • PHP_SESSION_NONE 会话是启用的，但不存在当前会话。
  • PHP_SESSION_ACTIVE 会话是启用的，而且存在当前会话。



参见

  • {@link session_start()} - 启动新会话或者重用现有会话
