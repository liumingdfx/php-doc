Exception::getTrace

(PHP 5, PHP 7, PHP 8)

Exception::getTrace — 获取异常追踪信息



说明

final public Exception::getTrace(): array

返回异常追踪信息。



参数

此函数没有参数。


返回值

返回包含异常追踪信息的数组（array）。



示例

示例 #1 Exception::getTrace()示例

<?php
function test() {
throw new Exception;
}

try {
test();
} catch(Exception $e) {
var_dump($e->getTrace());
}
?>

以上示例的输出类似于：

array(1) {
[0]=>
array(4) {
["file"]=>
string(22) "/home/bjori/tmp/ex.php"
["line"]=>
int(7)
["function"]=>
string(4) "test"
["args"]=>
array(0) {
}
}
}
