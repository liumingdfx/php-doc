swoole_async_dns_lookup

(PECL swoole >= 1.9.0)

swoole_async_dns_lookup — Async and non-blocking hostname to IP lookup



说明

swoole_async_dns_lookup(string $hostname, callable $callback): bool



参数

hostname
The host name.
callback
callback(string $hostname, string $ip): mixed
hostname

The host name.

IP

The IP address.



返回值

成功时返回 true， 或者在失败时返回 false。
