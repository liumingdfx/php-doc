The MongoDB\Driver\WriteError class

(mongodb >=1.0.0)

简介

The MongoDB\Driver\WriteError class encapsulates information about a write error and may be returned as an array element from {@link MongoDB\Driver\WriteResult::getWriteErrors()}.

类摘要

final class MongoDB\Driver\WriteError {
// 方法
final public getCode(): int
final public getIndex(): int
final public getInfo(): ?object
final public getMessage(): string
}

目录

  • {@link MongoDB\Driver\WriteError::getCode} — Returns the WriteError's error code
  • {@link MongoDB\Driver\WriteError::getIndex} — Returns the index of the write operation corresponding to this WriteError
  • {@link MongoDB\Driver\WriteError::getInfo} — Returns metadata document for the WriteError
  • {@link MongoDB\Driver\WriteError::getMessage} — Returns the WriteError's error message
