fann_save_train

(PECL fann >= 1.0.0)

fann_save_train — 将训练结构体保存至文件。



说明

fann_save_train(resource $data, string $file_name): bool

将训练结构体保存至文件， 格式和 {@link fann_read_train_from_file()} 函数中指定的一样。



参数

data
神经网络训练数据 资源。
file_name

保存训练数据的文件名。



返回值

成功时返回 true，其它情况下返回 false。



参见

  • {@link fann_read_train_from_file()} - 读取存储训练数据的文件。
