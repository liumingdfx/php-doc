ps_new

(PECL ps >= 1.1.0)

ps_new — Creates a new PostScript document object



说明

ps_new(): resource|false

Creates a new document instance. It does not create the file on disk or in memory, it just sets up everything. ps_new() is usually followed by a call of {@link ps_open_file()} to actually create the postscript document.



参数

此函数没有参数。


返回值

Resource of PostScript document 或者在失败时返回 false. The return value is passed to all other functions as the first argument.



参见

  • {@link ps_delete()} - Deletes all resources of a PostScript document
