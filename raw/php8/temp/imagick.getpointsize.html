Imagick::getPointSize

(PECL imagick 2 >= 2.1.0, PECL imagick 3)

Imagick::getPointSize — Gets point size



说明

public Imagick::getPointSize(): float

Returns the objects point size property. 此方法在Imagick基于ImageMagick 6.3.7以上版本编译时可用。



参数

此函数没有参数。


返回值

Returns a float containing the point size.



参见

  • {@link Imagick::setPointSize()} - Sets point size
