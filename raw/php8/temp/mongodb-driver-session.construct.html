MongoDB\Driver\Session::__construct

(mongodb >=1.4.0)

MongoDB\Driver\Session::__construct — Create a new Session (not used)



说明

final private MongoDB\Driver\Session::__construct()

MongoDB\Driver\Session objects are returned by {@link MongoDB\Driver\Manager::startSession()} and cannot be constructed directly.



参数

此函数没有参数。


参见

  • {@link MongoDB\Driver\Manager::startSession()} - Start a new client session for use with this client
