The SplSubject interface

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

简介

The SplSubject interface is used alongside SplObserver to implement the Observer Design Pattern.

接口摘要

interface SplSubject {
// 方法
public attach(SplObserver $observer): void
public detach(SplObserver $observer): void
public notify(): void
}

目录

  • {@link SplSubject::attach} — Attach an SplObserver
  • {@link SplSubject::detach} — Detach an observer
  • {@link SplSubject::notify} — Notify an observer
