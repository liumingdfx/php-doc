dbase_delete_record

(PHP 5 < 5.3.0, dbase 5, dbase 7)

dbase_delete_record — Deletes a record from a database



说明

dbase_delete_record(resource $database, int $number): bool

Marks the given record to be deleted from the database.

注意:

To actually remove the record from the database, you must also call {@link dbase_pack()}.



参数

database

The database resource, returned by {@link dbase_open()} or {@link dbase_create()}.

number

An integer which spans from 1 to the number of records in the database (as returned by {@link dbase_numrecords()}).



返回值

成功时返回 true， 或者在失败时返回 false。



更新日志

版本 说明
dbase 7.0.0 database is now a resource instead of an int.
