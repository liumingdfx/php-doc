DOMElement::removeAttributeNode

(PHP 5, PHP 7, PHP 8)

DOMElement::removeAttributeNode — Removes attribute



说明

public DOMElement::removeAttributeNode(DOMAttr $attr): DOMAttr|false

Removes attribute attr from the element.



参数

attr

The attribute node.



返回值

成功时返回 true， 或者在失败时返回 false。



错误／异常

DOM_NO_MODIFICATION_ALLOWED_ERR

Raised if the node is readonly.

DOM_NOT_FOUND_ERROR

Raised if attr is not an attribute of the element.



参见

  • {@link DOMElement::hasAttribute()} - Checks to see if attribute exists
  • {@link DOMElement::getAttributeNode()} - Returns attribute node
  • {@link DOMElement::setAttributeNode()} - Adds new attribute node to element
