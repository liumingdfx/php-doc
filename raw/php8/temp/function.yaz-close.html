yaz_close

(PHP 4 >= 4.0.1, PECL yaz >= 0.9.0)

yaz_close — Close YAZ connection



说明

yaz_close(resource $id): bool

Closes the connection given by parameter id.

注意:

This function will only close a non-persistent connection opened by setting the persistent option to false with {@link yaz_connect()}.



参数

id

The connection resource returned by {@link yaz_connect()}.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link yaz_connect()} - Prepares for a connection to a Z39.50 server
