mysqli::$protocol_version

mysqli_get_proto_info

(PHP 5, PHP 7, PHP 8)

mysqli::$protocol_version -- mysqli_get_proto_info — 返回 MySQL 协议使用的版本号



说明

面向对象风格

int $mysqli->protocol_version;

过程化风格

mysqli_get_proto_info(mysqli $mysql): int

返回整数，表示由 mysql 参数表示的连接所使用的 MySQL 协议的版本号。



参数

mysql

仅以过程化样式：由 {@link mysqli_connect()} 或mysqli_init() 返回的 mysqli 对象。


返回值

返回表示协议版本号的整数。



示例

示例 #1 $mysqli->protocol_version 示例

面向对象风格

<?php

mysqli_report(MYSQLI_REPORT_ERROR | MYSQLI_REPORT_STRICT);
$mysqli = new mysqli("localhost", "my_user", "my_password");

// print protocol version
printf("Protocol version: %d\n", $mysqli->protocol_version);

过程化风格

<?php

mysqli_report(MYSQLI_REPORT_ERROR | MYSQLI_REPORT_STRICT);
$link = mysqli_connect("localhost", "my_user", "my_password");

// print protocol version
printf("Protocol version: %d\n", mysqli_get_proto_info($link));

以上示例会输出：

Protocol version: 10
