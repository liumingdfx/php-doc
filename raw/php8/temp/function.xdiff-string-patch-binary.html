xdiff_string_patch_binary

(PECL xdiff >= 0.2.0)

xdiff_string_patch_binary — 别名 {@link xdiff_string_bpatch()}



说明

xdiff_string_patch_binary(string $str, string $patch): string

Patches a string str with a binary patch. This function accepts patches created both via {@link xdiff_string_bdiff()} and {@link xdiff_string_rabdiff()} functions or their file counterparts.

Starting with version 1.5.0 this function is an alias of {@link xdiff_string_bpatch()}.



参数

str

The original binary string.

patch

The binary patch string.



返回值

Returns the patched string, or false on error.



参见

  • {@link xdiff_string_bpatch()} - Patch a string with a binary diff
  • {@link xdiff_string_bdiff()} - Make binary diff of two strings
  • {@link xdiff_string_rabdiff()} - Make binary diff of two strings using the Rabin's polynomial fingerprinting algorithm
