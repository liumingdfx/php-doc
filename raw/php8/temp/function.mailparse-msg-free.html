mailparse_msg_free

(PECL mailparse >= 0.9.0)

mailparse_msg_free — Frees a MIME resource



说明

mailparse_msg_free(resource $mimemail): bool

Frees a MIME resource.



参数

mimemail

A valid MIME resource allocated by {@link mailparse_msg_create()} or {@link mailparse_msg_parse_file()}.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link mailparse_msg_create()} - Create a mime mail resource
  • {@link mailparse_msg_parse_file()} - Parses a file
