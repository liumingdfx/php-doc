imap_rfc822_parse_headers

(PHP 4, PHP 5, PHP 7, PHP 8)

imap_rfc822_parse_headers — Parse mail headers from a string



说明

imap_rfc822_parse_headers(string $headers, string $default_hostname = "UNKNOWN"): stdClass

Gets an object of various header elements, similar to {@link imap_header()}.



参数

headers

The parsed headers data

default_hostname

The default host name



返回值

Returns an object similar to the one returned by {@link imap_header()}, except for the flags and other properties that come from the IMAP server.
