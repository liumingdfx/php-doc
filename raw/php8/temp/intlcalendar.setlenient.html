IntlCalendar::setLenient

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::setLenient — Set whether date/time interpretation is to be lenient



说明

面向对象风格

public IntlCalendar::setLenient(bool $lenient): true

过程化风格

intlcal_set_lenient(IntlCalendar $calendar, bool $lenient): true

Defines whether the calendar is ‘lenient mode’. In such a mode, some of out-of-bounds values for some fields are accepted, the behavior being similar to that of {@link IntlCalendar::add()} (i.e., the value wraps around, carrying into more significant fields each time). If the lenient mode is off, then such values will generate an error.



参数

calendar
IntlCalendar 实例。
lenient

Use true to activate the lenient mode; false otherwise.



返回值

总是返回 true。



更新日志

版本 说明
8.2.0 现在返回类型为 true；之前是 bool。


示例

See the example in {@link IntlCalendar::isLenient()}.
