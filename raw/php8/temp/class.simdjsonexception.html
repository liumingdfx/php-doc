The SimdJsonException class

(PECL simdjson >= 2.1.0)

简介

Exception thrown if {@link simdjson_decode()}, {@link simdjson_key_count()}, {@link simdjson_key_exists()}, or {@link simdjson_key_value()}. For possible values see the simdjson error codes constants.

类摘要

class SimdJsonException extends RuntimeException {
// 继承的属性
protected string $message = "";
private string $string = "";
protected int $code;
protected string $file = "";
protected int $line;
private array $trace = [];
private ?Throwable $previous = null;
}
