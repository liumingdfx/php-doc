The MongoDB\Driver\WriteConcernError class

(mongodb >=1.0.0)

简介

The MongoDB\Driver\WriteConcernError class encapsulates information about a write concern error and may be returned by {@link MongoDB\Driver\WriteResult::getWriteConcernError()}.

类摘要

final class MongoDB\Driver\WriteConcernError {
// 方法
final public getCode(): int
final public getInfo(): ?object
final public getMessage(): string
}

目录

  • {@link MongoDB\Driver\WriteConcernError::getInfo} — Returns metadata document for the WriteConcernError
  • {@link MongoDB\Driver\WriteConcernError::getMessage} — Returns the WriteConcernError's error message
