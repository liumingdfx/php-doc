gnupg_geterrorinfo

(PECL gnupg >= 1.5)

gnupg_geterrorinfo — Returns the error info



说明

gnupg_geterrorinfo(resource $identifier): array


参数

identifier

gnupg 标识符，由对{@link gnupg_init()} 或 gnupg 的调用生成。


返回值

Returns an array with error info.



示例

示例 #1 Procedural gnupg_geterrorinfo() example

<?php
$res = gnupg_init();
// this is called without any error
print_r(gnupg_geterrorinfo($res));
?>

以上示例会输出：

array(4) {
["generic_message"]=>
bool(false)
["gpgme_code"]=>
int(0)
["gpgme_source"]=>
string(18) "Unspecified source"
["gpgme_message"]=>
string(7) "Success"
}

示例 #2 OO gnupg_geterrorinfo() example

<?php
$gpg = new gnupg();
// error call
$gpg->decrypt('abc');
// error info should be displayed
print_r($gpg->geterrorinfo());
?>

以上示例会输出：

array(4) {
["generic_message"]=>
string(14) "decrypt failed"
["gpgme_code"]=>
int(117440570)
["gpgme_source"]=>
string(5) "GPGME"
["gpgme_message"]=>
string(7) "No data"
}
