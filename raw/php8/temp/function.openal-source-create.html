openal_source_create

(PECL openal >= 0.1.0)

openal_source_create — Generate a source resource



说明

openal_source_create(): resource


参数

此函数没有参数。


返回值

Returns an Open AL(Source) resource on success or false on failure.



参见

  • {@link openal_source_set()} - Set source property
  • {@link openal_source_play()} - Start playing the source
  • {@link openal_source_destroy()} - Destroy a source resource
