pg_connection_reset

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

pg_connection_reset — 重置连接（再次连接）



说明

pg_connection_reset(PgSql\Connection $connection): bool

pg_connection_reset() 重置连接。用于错误恢复。



参数

connection

PgSql\Connection 实例。


返回值

成功时返回 true， 或者在失败时返回 false。



更新日志

版本 说明
8.1.0 现在 connection 参数接受 PgSql\Connection 实例，之前接受 resource。


示例

示例 #1 pg_connection_reset() 示例

<?php
$dbconn = pg_connect("dbname=publisher") or die("Could not connect");
$dbconn2 = pg_connection_reset($dbconn);
if ($dbconn2) {
echo "reset successful\n";
} else {
echo "reset failed\n";
}
?>


参见

  • {@link pg_connect()} - 打开 PostgreSQL 连接
  • {@link pg_pconnect()} - 打开一个持久的 PostgreSQL 连接
  • {@link pg_connection_status()} - 获取连接状态
