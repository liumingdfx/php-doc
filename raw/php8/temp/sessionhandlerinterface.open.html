SessionHandlerInterface::open

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

SessionHandlerInterface::open — Initialize session



说明

public SessionHandlerInterface::open(string $path, string $name): bool

Re-initialize existing session, or creates a new one. Called when a session starts or when {@link session_start()} is invoked.



参数

path
The path where to store/retrieve the session.
name

The session name.



返回值

会话存储的返回值（通常成功返回 0，失败返回 1）。



参见

  • {@link session_name()} - 读取/设置会话名称
  • The session.auto-start configuration directive.
