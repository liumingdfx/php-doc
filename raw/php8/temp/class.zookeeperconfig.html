The ZookeeperConfig class

(PECL zookeeper >= 0.6.0, ZooKeeper >= 3.5.0)

简介

The ZooKeeper Config handling class.

类摘要

class ZookeeperConfig {
// 方法
public add(string $members, int $version = -1, array &$stat = null): void
public get(callable $watcher_cb = null, array &$stat = null): string
public remove(string $id_list, int $version = -1, array &$stat = null): void
public set(string $members, int $version = -1, array &$stat = null): void
}

目录

  • {@link ZookeeperConfig::add} — Add servers to the ensemble
  • {@link ZookeeperConfig::get} — Gets the last committed configuration of the ZooKeeper cluster as it is known to the server to which the client is connected, synchronously
  • {@link ZookeeperConfig::remove} — Remove servers from the ensemble
  • {@link ZookeeperConfig::set} — Change ZK cluster ensemble membership and roles of ensemble peers
