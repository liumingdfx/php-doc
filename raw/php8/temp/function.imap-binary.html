imap_binary

(PHP 4, PHP 5, PHP 7, PHP 8)

imap_binary — Convert an 8bit string to a base64 string



说明

imap_binary(string $string): string|false

Convert an 8bit string to a base64 string according to » RFC2045, Section 6.8.



参数

string

The 8bit string



返回值

Returns a base64 encoded string, 或者在失败时返回 false.

  • {@link imap_base64()} - Decode BASE64 encoded text
