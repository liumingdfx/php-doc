apache_get_version

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

apache_get_version — 获得Apache版本信息



说明

apache_get_version(): string|false

获得Apache版本信息。



参数

此函数没有参数。


返回值

成功时返回 Apache 版本信息 或者在失败时返回 false.



示例

示例 #1 apache_get_version() 示例

<?php
$version = apache_get_version();
echo "$version\n";
?>

以上示例的输出类似于：

Apache/1.3.29 (Unix) PHP/4.3.4


参见

  • {@link phpinfo()} - 输出关于 PHP 配置的信息
