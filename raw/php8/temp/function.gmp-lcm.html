gmp_lcm

(PHP 7 >= 7.3.0, PHP 8)

gmp_lcm — Calculate LCM



说明

gmp_lcm(GMP|int|string $num1, GMP|int|string $num2): GMP

This function computes the least common multiple (lcm) of num1 and num2.



参数

num1
GMP 对象或 int ，或数字string。
num2

GMP 对象或 int ，或数字string。



返回值

GMP 对象。



参见

  • {@link gmp_gcd()} - Calculate GCD
