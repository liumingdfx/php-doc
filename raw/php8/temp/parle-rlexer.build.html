Parle\RLexer::build

(PECL parle >= 0.5.1)

Parle\RLexer::build — Finalize the lexer rule set



说明

public Parle\RLexer::build(): void

Rules, previously added with {@link Parle\RLexer::push()} are finalized. This method call has to be done after all the necessary rules was pushed. The rule set becomes read only. The lexing can begin.



参数

此函数没有参数。


返回值

没有返回值。
