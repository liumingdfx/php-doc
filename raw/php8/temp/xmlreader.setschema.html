XMLReader::setSchema

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

XMLReader::setSchema — Validate document against XSD



说明

public XMLReader::setSchema(?string $filename): bool

Use W3C XSD schema to validate the document as it is processed. Activation is only possible before the first Read().



参数

filename

The filename of the XSD schema.



返回值

成功时返回 true， 或者在失败时返回 false。



错误／异常

Issues E_WARNING if libxml was built without schema support, the schema contains errors or if {@link XMLReader::read()} has already been called.



注释

警告

此函数仅在 PHP 与 libxml 20620 或以上版本编译时可用。



参见

  • {@link XMLReader::setRelaxNGSchema()} - Set the filename or URI for a RelaxNG Schema
  • {@link XMLReader::setRelaxNGSchemaSource()} - Set the data containing a RelaxNG Schema
  • {@link XMLReader::isValid()} - Indicates if the parsed document is valid
