shm_detach

(PHP 4, PHP 5, PHP 7, PHP 8)

shm_detach — Disconnects from shared memory segment



说明

shm_detach(SysvSharedMemory $shm): bool

shm_detach() disconnects from the shared memory given by the shm created by {@link shm_attach()}. Remember, that shared memory still exist in the Unix system and the data is still present.



参数

shm

A shared memory segment obtained from {@link shm_attach()}.



返回值

成功时返回 true， 或者在失败时返回 false。



更新日志

版本 说明
8.0.0 shm expects a SysvSharedMemory instance now; previously, a resource was expected.


参见

  • {@link shm_attach()} - Creates or open a shared memory segment
  • {@link shm_remove()} - Removes shared memory from Unix systems
  • {@link shm_remove_var()} - Removes a variable from shared memory
