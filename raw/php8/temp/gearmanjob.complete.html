GearmanJob::complete

(PECL gearman <= 0.5.0)

GearmanJob::complete — Send the result and complete status (deprecated)



说明

public GearmanJob::complete(string $result): bool

Sends result data and the complete status update for this job.

注意:

This method has been replaced by {@link GearmanJob::sendComplete()} in the 0.6.0 release of the Gearman extension.



参数

result

Serialized result data.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link GearmanJob::sendFail()} - Send fail status
  • {@link GearmanJob::setReturn()} - Set a return value
