zip_open

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PHP 8, PECL zip >= 1.0.0)

zip_open — 打开 ZIP 文件归档

警告

本函数已自 PHP 8.0.0 起被废弃。强烈建议不要依赖本函数。



说明

zip_open(string $filename): resource|int|false

打开一个新的ZIP归档文件进行读取。



参数

filename

待打开ZIP归档的文件名。



返回值

成功的时候返回资源句柄供 {@link zip_read()} 和 {@link zip_close()} 后续使用；如果 filename 文件不存在或者出现其他错误，则会返回 false 或相应的错误码。



更新日志

版本 说明
8.0.0 弃用此函数，取而代之的是对象 API，请参阅 {@link ZipArchive::open()}。


参见

  • {@link zip_read()} - 读取 ZIP 文件归档中下一项
  • {@link zip_close()} - 关闭一个ZIP档案文件
