Imagick::getImageIndex

(PECL imagick 2, PECL imagick 3)

Imagick::getImageIndex — Gets the index of the current active image

警告

此函数在 Imagick 3.4.4 中被 废弃，强烈建议不要应用此函数。



说明

public Imagick::getImageIndex(): int

Returns the index of the current active image within the Imagick object. This method has been deprecated. See {@link Imagick::getIteratorIndex()}.



参数

此函数没有参数。


返回值

Returns an integer containing the index of the image in the stack.



错误／异常

错误时抛出 ImagickException。
