uopz_get_hook

(PECL uopz 5, PECL uopz 6, PECL uopz 7)

uopz_get_hook — Gets previously set hook on function or method



说明

uopz_get_hook(string $function): Closure
uopz_get_hook(string $class, string $function): Closure

Gets the previously set hook on a function or method.



参数

class
The name of the class.
function

The name of the function or method.



返回值

Returns the previously set hook on a function or method, or null if no hook has been set.



示例

示例 #1 Basic uopz_get_hook() Usage

<?php
function foo() {
echo 'foo';
}
uopz_set_hook('foo', function () {echo 'bar';});
var_dump(uopz_get_hook('foo'));
?>

以上示例的输出类似于：

object(Closure)#2 (0) {
}


参见

  • {@link uopz_set_hook()} - Sets hook to execute when entering a function or method
  • {@link uopz_unset_hook()} - Removes previously set hook on function or method
