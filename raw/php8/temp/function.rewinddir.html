rewinddir

(PHP 4, PHP 5, PHP 7, PHP 8)

rewinddir — 倒回目录句柄



说明

rewinddir(?resource $dir_handle = null): void

将 dir_handle 指定的目录流重置到目录的开头。



参数

dir_handle

之前使用 {@link opendir()} 打开的目录句柄 resource。如果未指定目录句柄，则假设是 {@link opendir()} 打开的最后一个链接。



返回值

没有返回值。



更新日志

版本 说明
8.0.0 现在 dir_handle 允许为 null。
