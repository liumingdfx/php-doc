The MongoDB\BSON\Persistable interface

(mongodb >=1.0.0)

简介

Classes may implement this interface to take advantage of automatic ODM (object document mapping) behavior in the driver. During serialization, the driver will inject a __pclass property containing the PHP class name into the data returned by {@link MongoDB\BSON\Serializable::bsonSerialize()}. During unserialization, the same __pclass property will then be used to infer the PHP class (independent of any type map configuration) to be constructed before {@link MongoDB\BSON\Unserializable::bsonUnserialize()} is invoked. See Persisting Data for additional information.

注意: Even if {@link MongoDB\BSON\Serializable::bsonSerialize()} would return a sequential array, injection of the __pclass property will cause the object to be serialized as a BSON document.

接口摘要

class MongoDB\BSON\Persistable implements MongoDB\BSON\Unserializable, MongoDB\BSON\Serializable {
// 方法
abstract public bsonSerialize(): array|stdClass|MongoDB\BSON\Document
// 继承的方法
abstract public {@link MongoDB\BSON\Serializable::bsonSerialize}(): array|stdClass|MongoDB\BSON\Document|MongoDB\BSON\PackedArray
abstract public {@link MongoDB\BSON\Unserializable::bsonUnserialize}(array $data): void
}
