dbase_close

(PHP 5 < 5.3.0, dbase 5, dbase 7)

dbase_close — Closes a database



说明

dbase_close(resource $database): bool

Closes the given database resource.



参数

database

The database resource, returned by {@link dbase_open()} or {@link dbase_create()}.



返回值

成功时返回 true， 或者在失败时返回 false。



更新日志

版本 说明
dbase 7.0.0 database is now a resource instead of an int.


示例

示例 #1 Closing a dBase database file

<?php

// open in read-only mode
$db = dbase_open('/tmp/test.dbf', 0);

if ($db) {
// read some data ..

dbase_close($db);
}

?>


参见

  • {@link dbase_open()} - Opens a database
  • {@link dbase_create()} - Creates a database
