runkit7_function_remove

(PECL runkit7 >= Unknown)

runkit7_function_remove — Remove a function definition



说明

runkit7_function_remove(string $function_name): bool

注意: 默认情况下，仅在用户空间可删除，重命名，或者修改函数。为了覆盖内部函数，必须启用 php.ini 中的runkit.internal_override 设置。



参数

function_name
Name of the function to be deleted


返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link runkit7_function_add()} - Add a new function, similar to create_function
  • {@link runkit7_function_copy()} - Copy a function to a new function name
  • {@link runkit7_function_redefine()} - Replace a function definition with a new implementation
  • {@link runkit7_function_rename()} - Change a function's name
