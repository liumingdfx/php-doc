Reflection::export

(PHP 5, PHP 7)

Reflection::export — 导出

警告

本函数已自 PHP 7.4.0起弃用，自 PHP 8.0.0 起移除。强烈建议不要依赖本函数。



说明

public static Reflection::export(Reflector $reflector, bool $return = false): string

导出反射（reflection）。

警告

本函数还未编写文档，仅有参数列表。



参数

reflector

导出的反射。

return

设为 true 时返回导出结果，设为 false（默认值）则忽略返回。



返回值

如果参数 return 设为 true，导出结果将作为 string返回，否则返回 null。



参见

  • {@link Reflection::getModifierNames()} - 获取修饰符的名称
