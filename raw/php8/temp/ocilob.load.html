OCILob::load

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCILob::load — Returns large object's contents



说明

public OCILob::load(): string|false

Returns large object's contents. As script execution is terminated when the memory_limit is reached, ensure that the LOB does not exceed this limit. In most cases it's recommended to use {@link OCILob::read} instead.



参数

此函数没有参数。


返回值

Returns the contents of the object, or false on errors.

版本 说明
