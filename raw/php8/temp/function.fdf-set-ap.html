fdf_set_ap

(PHP 4, PHP 5 < 5.3.0, PECL fdf SVN)

fdf_set_ap — Set the appearance of a field



说明

fdf_set_ap(
    resource $fdf_document,
    string $field_name,
    int $face,
    string $filename,
    int $page_number
): bool

Sets the appearance of a field (i.e. the value of the /AP key).



参数

fdf_document

The FDF document handle, returned by {@link fdf_create()}, {@link fdf_open()} or {@link fdf_open_string()}.

field_name

face

The possible values FDFNormalAP, FDFRolloverAP and FDFDownAP.

filename

page_number


返回值

成功时返回 true， 或者在失败时返回 false。
