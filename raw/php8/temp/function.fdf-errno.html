fdf_errno

(PHP 4 >= 4.3.0, PHP 5 < 5.3.0, PECL fdf SVN)

fdf_errno — Return error code for last fdf operation



说明

fdf_errno(): int

Gets the error code set by the last FDF function call.

A textual description of the error may be obtained using with {@link fdf_error()}.



参数

此函数没有参数。


返回值

Returns the error code as an integer, or zero if there was no errors.



参见

  • {@link fdf_error()} - Return error description for FDF error code
