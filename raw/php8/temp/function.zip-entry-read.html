zip_entry_read

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PHP 8, PECL zip >= 1.0.0)

zip_entry_read — 读取一个打开了的压缩目录实体

警告

本函数已自 PHP 8.0.0 起被废弃。强烈建议不要依赖本函数。



说明

zip_entry_read(resource $zip_entry, int $len = 1024): string|false

读取一个打开了的压缩目录实体。



参数

zip_entry

由函数{@link zip_read()} 返回的目录实体。

len

需要返回的字节数。

注意:

这字节数应该是你所要读取的未压缩的字节数。



更新日志

版本 说明
8.0.0 弃用此函数，取而代之的是对象 API，请参阅 {@link ZipArchive::getFromIndex()}。


返回值

成功的时候返回读取到的数据；到达文件末尾的时候返回一个空的字符串； 读取出错的时候则会返回false


参见

  • {@link zip_entry_open()} - 打开用于读取的目录实体
  • {@link zip_entry_close()} - 关闭目录项
  • {@link zip_entry_filesize()} - 检索目录实体的实际大小
