The EmptyIterator class

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

简介

The EmptyIterator class for an empty iterator.

类摘要

class EmptyIterator implements Iterator {
// 方法
public current(): never
public key(): never
public next(): void
public rewind(): void
public valid(): false
}
