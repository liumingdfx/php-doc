ReflectionEnum::isBacked

(PHP 8 >= 8.1.0)

ReflectionEnum::isBacked — 检测 Enum 是否为回退（Backed）Enum



说明

public ReflectionEnum::isBacked(): bool

回退 Enum 是具有本地 string 和 int 回退标量值。 并非所有枚举都是回退的。



参数

此函数没有参数。


返回值

如果枚举具有回退标量，返回 true，否则返回 false。



示例

示例 #1 ReflectionEnum::isBacked() 示例

<?php
enum Suit
{
case Hearts;
case Diamonds;
case Clubs;
case Spades;
}

enum BackedSuit: string
{
case Hearts = 'H';
case Diamonds = 'D';
case Clubs = 'C';
case Spades = 'S';
}

var_dump((new ReflectionEnum(Suit::class))->isBacked());
var_dump((new ReflectionEnum(BackedSuit::class))->isBacked());
?>

以上示例会输出：

bool(false)
bool(true)
