tan

(PHP 4, PHP 5, PHP 7, PHP 8)

tan — 正切



说明

tan(float $num): float

tan() 返回参数 num 的正切值。参数 num 的单位为弧度。



参数

num

要处理的以弧度为单位的参数



返回值

num 的正切值



示例

示例 #1 tan() 示例

<?php

echo tan(M_PI_4); // 1

?>

  • {@link atan2()} - 两个参数的反正切
  • {@link sin()} - 正弦
  • {@link cos()} - 余弦
  • {@link tanh()} - 双曲正切
  • {@link deg2rad()} - 将角度转换为弧度
