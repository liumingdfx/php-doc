eio_nready

(PECL eio >= 0.0.1dev)

eio_nready — Returns number of not-yet handled requests



说明

eio_nready(): int



参数

此函数没有参数。


返回值

eio_nready() returns number of not-yet handled requests



参见

  • {@link eio_nreqs()} - Returns number of requests to be processed
  • eio_nready()
  • {@link eio_nthreads()} - Returns number of threads currently in use
