DOMElement::hasAttribute

(PHP 5, PHP 7, PHP 8)

DOMElement::hasAttribute — Checks to see if attribute exists



说明

public DOMElement::hasAttribute(string $qualifiedName): bool

Indicates whether attribute named qualifiedName exists as a member of the element.



参数

qualifiedName

The attribute name.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link DOMElement::hasAttributeNS()} - Checks to see if attribute exists
  • {@link DOMElement::getAttribute()} - Returns value of attribute
  • {@link DOMElement::setAttribute()} - Adds new or modifies existing attribute
  • {@link DOMElement::removeAttribute()} - Removes attribute
