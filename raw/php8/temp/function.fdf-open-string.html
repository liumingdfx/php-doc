fdf_open_string

(PHP 4 >= 4.3.0, PHP 5 < 5.3.0, PECL fdf SVN)

fdf_open_string — Read a FDF document from a string



说明

fdf_open_string(string $fdf_data): resource

Reads form data from a string.

You can use fdf_open_string() together with $HTTP_FDF_DATA to process FDF form input from a remote client.



参数

fdf_data

The data as returned from a PDF form or created using {@link fdf_create()} and {@link fdf_save_string()}.



返回值

Returns a FDF document handle, or false on error.



示例

示例 #1 Accessing the form data

<?php
$fdf = fdf_open_string($HTTP_FDF_DATA);
// ...
fdf_close($fdf);
?>


参见

  • {@link fdf_open()} - Open a FDF document
  • {@link fdf_close()} - Close an FDF document
  • {@link fdf_create()} - Create a new FDF document
  • {@link fdf_save_string()} - Returns the FDF document as a string
