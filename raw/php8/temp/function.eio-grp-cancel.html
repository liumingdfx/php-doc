eio_grp_cancel

(PECL eio >= 0.0.1dev)

eio_grp_cancel — Cancels a request group



说明

eio_grp_cancel(resource $grp): void

eio_grp_cancel() cancels a group request specified by grp request group resource.



参数

grp
The request group resource returned by {@link eio_grp()}.


返回值

没有返回值。



参见

  • {@link eio_grp()} - Creates a request group
  • {@link eio_grp_add()} - Adds a request to the request group
