DOMNode::isSupported

(PHP 5, PHP 7, PHP 8)

DOMNode::isSupported — Checks if feature is supported for specified version



说明

public DOMNode::isSupported(string $feature, string $version): bool

Checks if the asked feature is supported for the specified version.



参数

feature

The feature to test. See the example of {@link DOMImplementation::hasFeature()} for a list of features.

version

The version number of the feature to test.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link DOMImplementation::hasFeature()} - Test if the DOM implementation implements a specific feature
