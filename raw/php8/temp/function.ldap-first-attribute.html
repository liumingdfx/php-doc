ldap_first_attribute

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_first_attribute — Return first attribute



说明

ldap_first_attribute(LDAP\Connection $ldap, LDAP\ResultEntry $entry): string|false

Gets the first attribute in the given entry. Remaining attributes are retrieved by calling {@link ldap_next_attribute()} successively.

Similar to reading entries, attributes are also read one by one from a particular entry.



参数

ldap

通过 {@link ldap_connect()} 返回的 LDAP\Connection 实例。

entry

LDAP\ResultEntry 实例。



返回值

Returns the first attribute in the entry on success and false on error.
