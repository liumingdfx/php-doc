SessionHandlerInterface::close

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

SessionHandlerInterface::close — Close the session



说明

public SessionHandlerInterface::close(): bool

Closes the current session. This function is automatically executed when closing the session, or explicitly via {@link session_write_close()}.



参数

此函数没有参数。


返回值

会话存储的返回值（通常成功返回 0，失败返回 1）。
