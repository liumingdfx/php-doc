imap_expunge

(PHP 4, PHP 5, PHP 7, PHP 8)

imap_expunge — Delete all messages marked for deletion



说明

imap_expunge(IMAP\Connection $imap): true

Deletes all the messages marked for deletion by {@link imap_delete()}, {@link imap_mail_move()}, or {@link imap_setflag_full()}.



参数

imap

IMAP\Connection 实例。


返回值

总是返回 true。



更新日志

版本 说明
8.1.0 现在 imap 参数接受 IMAP\Connection 实例，之前接受有效的 imap resource。
