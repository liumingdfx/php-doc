jdtounix

(PHP 4, PHP 5, PHP 7, PHP 8)

jdtounix — 将儒略日数转换为 Unix 时间戳



说明

jdtounix(int $julian_day): int

这个函数将返回与 julian_day 给出的儒略日相对应的 Unix 时间戳。返回的时间是 UTC。



参数

julian_day

儒略日数在 64 位系统上是 2440588 到 106751993607888 之间的数字，在 32 位系统上是 2440588 到 2465443 之间的数字。



返回值

指定儒略日的开始时（午夜，而非中午）的 unix 时间戳。



错误／异常

julian_day 超出允许的范围值，抛出 ValueError。



更新日志

版本 说明
