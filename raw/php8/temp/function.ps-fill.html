ps_fill

(PECL ps >= 1.1.0)

ps_fill — Fills the current path



说明

ps_fill(resource $psdoc): bool

Fills the path constructed with previously called drawing functions like {@link ps_lineto()}.



参数

psdoc

Resource identifier of the postscript file as returned by {@link ps_new()}.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link ps_fill_stroke()} - Fills and strokes the current path
  • {@link ps_stroke()} - Draws the current path
