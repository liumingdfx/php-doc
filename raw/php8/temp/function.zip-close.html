zip_close

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PHP 8, PECL zip >= 1.0.0)

zip_close — 关闭一个ZIP档案文件

警告

本函数已自 PHP 8.0.0 起被废弃。强烈建议不要依赖本函数。



说明

zip_close(resource $zip): void

关闭一个指定的ZIP档案文件。



参数

zip

一个由{@link zip_open()}打开的ZIP文件资源。



返回值

没有返回值。



更新日志

版本 说明
8.0.0 弃用此函数，取而代之的是对象 API，请参阅 {@link ZipArchive::close()}。


参见

  • {@link zip_open()} - 打开 ZIP 文件归档
  • {@link zip_read()} - 读取 ZIP 文件归档中下一项
