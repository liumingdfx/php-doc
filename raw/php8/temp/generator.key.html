Generator::key

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

Generator::key — 返回当前产生的键



说明

public Generator::key(): mixed

获取产生的值的键



参数

此函数没有参数。


返回值

返回当前产生的键。



示例

示例 #1 Generator::key() example

<?php

function Gen()
{
yield 'key' => 'value';
}

$gen = Gen();

echo "{$gen->key()} => {$gen->current()}";

以上示例会输出：

key => value
