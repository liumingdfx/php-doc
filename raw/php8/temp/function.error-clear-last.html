error_clear_last

(PHP 7, PHP 8)

error_clear_last — 清除最近一次错误



说明

error_clear_last(): void



参数

此函数没有参数。


返回值

清除最近一次错误，使它无法通过 {@link error_get_last()} 获取。



示例

示例 #1 error_clear_last() 例子

<?php
var_dump(error_get_last());
error_clear_last();
var_dump(error_get_last());

@$a = $b;

var_dump(error_get_last());
error_clear_last();
var_dump(error_get_last());
?>

以上示例的输出类似于：
