The Parle\ErrorInfo class

(PECL parle >= 0.5.2)

简介

The class represents detailed error information as supplied by {@link Parle\Parser::errorInfo()}

类摘要

class Parle\ErrorInfo {
// 属性
public int $id;
public int $position;
public mixed $token;
// 方法
}

属性

id

Error id.

position

Position in the input, where the error occurred.

token

If applicable - the Parle\Token related to the error, otherwise null.
