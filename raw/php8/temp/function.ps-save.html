ps_save

(PECL ps >= 1.1.0)

ps_save — Save current context



说明

ps_save(resource $psdoc): bool

Saves the current graphics context, containing colors, translation and rotation settings and some more. A saved context can be restored with {@link ps_restore()}.



参数

psdoc

Resource identifier of the postscript file as returned by {@link ps_new()}.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link ps_restore()} - Restore previously save context
