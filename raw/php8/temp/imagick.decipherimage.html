Imagick::decipherImage

(PECL imagick 2 >= 2.3.0, PECL imagick 3)

Imagick::decipherImage — Deciphers an image



说明

public Imagick::decipherImage(string $passphrase): bool

Deciphers image that has been enciphered before. The image must be enciphered using {@link Imagick::encipherImage()}. 此方法在Imagick基于ImageMagick 6.3.9以上版本编译时可用。



参数

passphrase

The passphrase



返回值

成功时返回 true。



参见

  • {@link Imagick::encipherImage()} - Enciphers an image
