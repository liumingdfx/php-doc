Creating Phar Archives: Introduction

To be written fully in the near future. Before reading this, be sure to read How to use Phar Archives.

A great place to start is by reading about {@link Phar::buildFromIterator()}, and the specifics of the file format choices available for archives. A healthy understanding of what a stub is and does is crucial to phar archive creation, and so {@link Phar::setStub()} and {@link Phar::createDefaultStub()} are good places to start as well. If you are distributing a web-based application, it is crucial to know about {@link Phar::webPhar()} and related method {@link Phar::mungServer()}. Any application that accesses its own files should also consider using {@link Phar::interceptFileFuncs()}.
