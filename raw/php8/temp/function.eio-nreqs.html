eio_nreqs

(PECL eio >= 0.0.1dev)

eio_nreqs — Returns number of requests to be processed



说明

eio_nreqs(): int

eio_nreqs() could be called in a custom loop calling {@link eio_poll()}.



参数

此函数没有参数。


返回值

eio_nreqs() returns number of requests to be processed.



示例

示例 #1 eio_nreqs() example

<?php
function res_cb($data, $result) {
var_dump($data);
var_dump($result);
}

eio_nop(EIO_PRI_DEFAULT, "res_cb", "1");
eio_nop(EIO_PRI_DEFAULT, "res_cb", "2");
eio_nop(EIO_PRI_DEFAULT, "res_cb", "3");

while (eio_nreqs()) {
eio_poll();
}
?>

以上示例的输出类似于：

string(1) "1"
int(0)
string(1) "3"
int(0)
string(1) "2"
int(0)


参见

  • {@link eio_poll()} - Can be to be called whenever there are pending requests that need finishing
  • {@link eio_nready()} - Returns number of not-yet handled requests
