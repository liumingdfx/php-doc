ps_end_page

(PECL ps >= 1.1.0)

ps_end_page — End a page



说明

ps_end_page(resource $psdoc): bool

Ends a page which was started with {@link ps_begin_page()}. Ending a page will leave the current drawing context, which e.g. requires to reload fonts if they were loading within the page, and to set many other drawing parameters like the line width, or color.



参数

psdoc

Resource identifier of the postscript file as returned by {@link ps_new()}.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link ps_begin_page()} - Start a new page
