ps_shfill

(PECL ps >= 1.3.0)

ps_shfill — Fills an area with a shading



说明

ps_shfill(resource $psdoc, int $shadingid): bool

Fills an area with a shading, which has to be created before with {@link ps_shading()}. This is an alternative way to creating a pattern from a shading {@link ps_shading_pattern()} and using the pattern as the filling color.



参数

psdoc

Resource identifier of the postscript file as returned by {@link ps_new()}.

shadingid

The identifier of a shading previously created with {@link ps_shading()}.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link ps_shading()} - Creates a shading for later use
  • {@link ps_shading_pattern()} - Creates a pattern based on a shading
