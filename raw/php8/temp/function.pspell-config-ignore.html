pspell_config_ignore

(PHP 4 >= 4.0.2, PHP 5, PHP 7, PHP 8)

pspell_config_ignore — Ignore words less than N characters long



说明

pspell_config_ignore(PSpell\Config $config, int $min_length): bool

pspell_config_ignore() should be used on a config before calling {@link pspell_new_config()}. This function allows short words to be skipped by the spell checker.



参数

config

An PSpell\Config instance.

min_length

Words less than min_length characters will be skipped.



返回值

成功时返回 true， 或者在失败时返回 false。



更新日志

版本 说明
8.1.0 The config parameter expects an PSpell\Config instance now; previously, a resource was expected.


示例

示例 #1 pspell_config_ignore()

<?php
$pspell_config = pspell_config_create("en");
pspell_config_ignore($pspell_config, 5);
$pspell = pspell_new_config($pspell_config);
pspell_check($pspell, "abcd"); //will not result in an error
?>
