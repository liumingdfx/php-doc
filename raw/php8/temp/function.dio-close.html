dio_close

(PHP 4 >= 4.2.0, PHP 5 < 5.1.0)

dio_close — 通过 fd 关闭文件描述符



说明

dio_close(resource $fd): void

dio_close() 函数关闭 fd 文件描述符。



参数

fd

{@link dio_open()} 返回的文件描述符。



返回值

没有返回值。



示例

示例 #1 关闭文件描述符

<?php
$fd = dio_open('/dev/ttyS0', O_RDWR);

dio_close($fd);
?>


参见

  • {@link dio_open()} - 在 C 库输入/输出流函数允许的更低级别打开（必要时创建）文件
