ZipArchive::clearError

(PHP 8 >= 8.2.0, PECL zip >= 1.20.0)

ZipArchive::clearError — 清除状态错误消息、系统和/或 zip 消息



说明

public ZipArchive::clearError(): void

清除状态错误消息、系统和/或 zip 消息。



参数

此函数没有参数。


返回值

没有返回值。



参见

  • {@link ZipArchive::getStatusString()} - Returns the status error message, system and/or zip messages
