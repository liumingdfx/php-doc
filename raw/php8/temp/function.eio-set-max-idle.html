eio_set_max_idle

(PECL eio >= 0.0.1dev)

eio_set_max_idle — Set maximum number of idle threads



说明

eio_set_max_idle(int $nthreads): void



参数

nthreads
Number of idle threads.


返回值

没有返回值。



参见

  • {@link eio_nthreads()} - Returns number of threads currently in use
  • {@link eio_set_min_parallel()} - Set minimum parallel thread number
  • {@link eio_set_max_parallel()} - Set maximum parallel threads
