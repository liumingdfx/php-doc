ps_delete

(PECL ps >= 1.1.0)

ps_delete — Deletes all resources of a PostScript document



说明

ps_delete(resource $psdoc): bool

Mainly frees memory used by the document. Also closes a file, if it was not closed before with {@link ps_close()}. You should in any case close the file with {@link ps_close()} before, because {@link ps_close()} not just closes the file but also outputs a trailor containing PostScript comments like the number of pages in the document and adding the bookmark hierarchy.



参数

psdoc

Resource identifier of the postscript file as returned by {@link ps_new()}.



返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link ps_close()} - Closes a PostScript document
