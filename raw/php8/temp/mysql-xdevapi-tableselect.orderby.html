TableSelect::orderby — Set select sort criteria



说明

public mysql_xdevapi\TableSelect::orderby(mixed $sort_expr, mixed ...$sort_exprs): mysql_xdevapi\TableSelect

Sets the order by criteria.



参数

sort_expr
The expressions that define the order by criteria. Can be an array with one or more expressions, or a string.
sort_exprs

Additional sort_expr parameters.



返回值

A TableSelect object.



示例

示例 #1 mysql_xdevapi\TableSelect::orderBy() example

<?php
$session = mysql_xdevapi\getSession("mysqlx://user:password@localhost");

$schema = $session->getSchema("addressbook");
$table = $schema->getTable("names");

$result = $table->select('name', 'age')
->orderBy('name desc')
->execute();

$row = $result->fetchAll();
print_r($row);
?>
