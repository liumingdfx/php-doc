yaz_get_option

(PECL yaz >= 0.9.0)

yaz_get_option — Returns value of option for connection



说明

yaz_get_option(resource $id, string $name): string

Returns the value of the option specified with name.



参数

id

The connection resource returned by {@link yaz_connect()}.

name

The option name.



返回值

Returns the value of the specified option or an empty string if the option wasn't set.



参见

  • The description of {@link yaz_set_option()} - Sets one or more options for connection for available options
