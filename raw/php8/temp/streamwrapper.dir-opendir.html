streamWrapper::dir_opendir

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

streamWrapper::dir_opendir — Open directory handle



说明

public streamWrapper::dir_opendir(string $path, int $options): bool

This method is called in response to {@link opendir()}.



参数

path

Specifies the URL that was passed to {@link opendir()}.

注意:

The URL can be broken apart with {@link parse_url()}.

options


返回值

成功时返回 true， 或者在失败时返回 false。



参见

  • {@link opendir()} - 打开目录句柄
  • {@link streamWrapper::dir_closedir()} - Close directory handle
  • {@link parse_url()} - 解析 URL，返回其组成部分
