Ds\Pair::copy — Returns a shallow copy of the pair



说明

public Ds\Pair::copy(): Ds\Pair

Returns a shallow copy of the pair.



参数

此函数没有参数。


返回值

Returns a shallow copy of the pair.



示例

示例 #1 Ds\Pair::copy() example

<?php
$a = new \Ds\Pair("a", 1);
$b = $a->copy();

$a->key = "x";

print_r($a);
print_r($b);
?>

以上示例的输出类似于：

Ds\Pair Object
(
[key] => x
[value] => 1
)
Ds\Pair Object
(
[key] => a
[value] => 1
)
