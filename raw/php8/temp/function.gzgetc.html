gzgetc

(PHP 4, PHP 5, PHP 7, PHP 8)

gzgetc — Get character from gz-file pointer



说明

gzgetc(resource $stream): string|false

Returns a string containing a single (uncompressed) character read from the given gz-file pointer.



参数

stream

The gz-file pointer. It must be valid, and must point to a file successfully opened by {@link gzopen()}.



返回值

The uncompressed character or false on EOF (unlike {@link gzeof()}).



示例

示例 #1 gzgetc() example

<?php
$gz = gzopen('somefile.gz', 'r');
while (!gzeof($gz)) {
echo gzgetc($gz);
}
gzclose($gz);
?>


参见

  • {@link gzopen()} - Open gz-file
