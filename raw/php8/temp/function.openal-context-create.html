openal_context_create

(PECL openal >= 0.1.0)

openal_context_create — Create an audio processing context



说明

openal_context_create(resource $device): resource


参数

device
An Open AL(Device) resource (previously created by {@link openal_device_open()}).


返回值

Returns an Open AL(Context) resource on success or false on failure.



参见

  • {@link openal_device_open()} - Initialize the OpenAL audio layer
  • {@link openal_context_destroy()} - Destroys a context
