Ds\Set::capacity

(PECL ds >= 1.0.0)

Ds\Set::capacity — Returns the current capacity



说明

public Ds\Set::capacity(): int

Returns the current capacity.



参数

此函数没有参数。


返回值

The current capacity.



示例

示例 #1 Ds\Set::capacity() example

<?php
$set = new \Ds\Set();
var_dump($set->capacity());

$set->add(...range(1, 50));
var_dump($set->capacity());
?>

以上示例的输出类似于：

int(16)
int(64)
