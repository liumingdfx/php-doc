radius_get_tagged_attr_data

(PECL radius >= 1.3.0)

radius_get_tagged_attr_data — Extracts the data from a tagged attribute



说明

radius_get_tagged_attr_data(string $data): string|false

If a tagged attribute has been returned from {@link radius_get_attr()}, radius_get_tagged_attr_data() will return the data from the attribute.



参数

data

The tagged attribute to be decoded.



返回值

Returns the data from the tagged attribute 或者在失败时返回 false.



示例

示例 #1 radius_get_tagged_attr_data() example

<?php
while ($resa = radius_get_attr($res)) {
if (!is_array($resa)) {
printf ("Error getting attribute: %s\n", radius_strerror($res));
exit;
}

$attr = $resa['attr'];
$data = $resa['data'];

$tag = radius_get_tagged_attr_tag($data);
$value = radius_get_tagged_attr_data($data);

printf("Got tagged attribute with tag %d and value %s\n", $tag, $value);
}
?>


参见

  • {@link radius_get_attr()} - Extracts an attribute
  • {@link radius_get_tagged_attr_tag()} - Extracts the tag from a tagged attribute
