radius_close

(PECL radius >= 1.1.0)

radius_close — Frees all ressources



说明

radius_close(resource $radius_handle): bool

It is not needed to call this function because php frees all resources at the end of each request.



参数

radius_handle

The RADIUS resource.


返回值

成功时返回 true， 或者在失败时返回 false。
