rrd_restore

(PECL rrd >= 0.9.0)

rrd_restore — 从 XML 转储中恢复 RRD 文件



说明

rrd_restore(string $xml_file, string $rrd_file, array $options = ?): bool

从 XML 存储中恢复 RRD 文件。



参数

xml_file
带有原始 RRD 数据库文件转储的 XML 文件名。
rrd_file

要恢复的 RRD 数据库文件名称。

options

恢复的选项数组。详情见 rrd restore 手册页。



返回值

成功时返回 true， 或者在失败时返回 false。
