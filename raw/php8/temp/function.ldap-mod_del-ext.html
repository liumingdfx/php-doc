ldap_mod_del_ext

(PHP 7 >= 7.3.0, PHP 8)

ldap_mod_del_ext — Delete attribute values from current attributes



说明

ldap_mod_del_ext(
    LDAP\Connection $ldap,
    string $dn,
    array $entry,
    ?array $controls = null
): LDAP\Result|false

Does the same thing as {@link ldap_mod_del()} but returns an LDAP\Result instance to be parsed with {@link ldap_parse_result()}.



参数

See {@link ldap_mod_del()}


返回值

Returns an LDAP\Result instance, 或者在失败时返回 false.



更新日志

版本 说明
