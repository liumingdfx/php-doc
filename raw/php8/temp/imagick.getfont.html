Imagick::getFont

(PECL imagick 2 >= 2.1.0, PECL imagick 3)

Imagick::getFont — Gets font



说明

public Imagick::getFont(): string

Returns the objects font property. 此方法在Imagick基于ImageMagick 6.3.7以上版本编译时可用。



参数

此函数没有参数。


返回值

Returns the string containing the font name or false if not font is set.



参见

  • {@link Imagick::setFont()} - Sets font
  • {@link ImagickDraw::setFont()} - Sets the fully-specified font to use when annotating with text
  • {@link ImagickDraw::getFont()} - Returns the font
