sodium_crypto_stream_xchacha20_keygen

(PHP 8 >= 8.1.0)

sodium_crypto_stream_xchacha20_keygen — Returns a secure random key



说明

sodium_crypto_stream_xchacha20_keygen(): string

Returns a secure random key for use with {@link sodium_crypto_stream_xchacha20()}.



参数

此函数没有参数。


返回值

Returns a 32-byte secure random key for use with {@link sodium_crypto_stream_xchacha20()}.
