ssh2_sftp_chmod

(PECL ssh2 >= 0.12)

ssh2_sftp_chmod — Changes file mode



说明

ssh2_sftp_chmod(resource $sftp, string $filename, int $mode): bool

Attempts to change the mode of the specified file to that given in mode.



参数

sftp

An SSH2 SFTP resource opened by {@link ssh2_sftp()}.

filename

Path to the file.

mode

Permissions on the file. See the {@link chmod()} for more details on this parameter.



返回值

成功时返回 true， 或者在失败时返回 false。



示例

示例 #1 Changing the mode of a file on a remote server

<?php
$connection = ssh2_connect('shell.example.com', 22);
ssh2_auth_password($connection, 'username', 'password');
$sftp = ssh2_sftp($connection);

ssh2_sftp_chmod($sftp, '/somedir/somefile', 0755);
?>
