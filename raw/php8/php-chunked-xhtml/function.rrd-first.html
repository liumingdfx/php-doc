rrd_first

(PECL rrd >= 0.9.0)

rrd_first — 从 rrd 文件中获取第一个样本的时间戳

说明

rrd_first(string $file, int $raaindex = 0): int

从 RRD 文件中指定的 RRA 返回第一个数据样本。

参数

file

RRD 数据库文件名。

raaindex

要检索的 RRA 索引。默认值为 0。

返回值

Unix 时间戳整数， 或者在失败时返回 false。
