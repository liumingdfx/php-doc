Yaf_Route_Map 类

(Yaf >=1.0.0)

简介

Yaf_Route_Map 是内置路由，它只是将 URI 端点（位于基础 URI 之后的 URI 部分：参阅 Yaf_Request_Abstract::setBaseUri()）转换为 controller 名或 action 名（取决于传递给 Yaf_Route_Map::__construct() 的参数），遵循以下规则： A => controller A。 A/B/C => controller A_B_C。 A/B/C/D/E => controller A_B_C_D_E。

如果指定 Yaf_Route_Map::__construct() 的第二个参数，则只有 URI 之前的部分用于路由，URI 之后的部分用于路由请求参数（参考 Yaf_Route_Map::__construct() 的示例部分）。
