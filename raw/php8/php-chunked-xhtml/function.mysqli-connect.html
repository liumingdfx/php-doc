mysqli_connect

(PHP 5, PHP 7, PHP 8)

mysqli_connect — 别名 mysqli::__construct()

说明

此函数是该函数的别名： mysqli::__construct()

注意:

如果未启用 mysqli 异常模式，并且连接失败，则 mysqli_connect() 返回 false 而不是对象。mysqli_connect_error() 函数可用于获取连接错误。
