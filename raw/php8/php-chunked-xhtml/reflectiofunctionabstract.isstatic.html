ReflectionFunctionAbstract::isStatic

(PHP 5, PHP 7, PHP 8)

ReflectionFunctionAbstract::isStatic — 判断函数是否为 static

说明

public ReflectionFunctionAbstract::isStatic(): bool

判断函数是否是 static。

参数

此函数没有参数。

返回值

函数为 static，返回 true，否则为 false

更新日志

版本 说明
8.1.0 此方法已提升。之前仅定义于 ReflectionMethod。
