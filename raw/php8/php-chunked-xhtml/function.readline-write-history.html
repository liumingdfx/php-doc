readline_write_history

(PHP 4, PHP 5, PHP 7, PHP 8)

readline_write_history — 写入历史记录

说明

readline_write_history(?string $filename = null): bool

这个函数将命令历史写入到文件。

参数

filename

保存文件的路径.

返回值

成功时返回 true， 或者在失败时返回 false。

更新日志

版本 说明
8.0.0 filename 现在可为 null。
