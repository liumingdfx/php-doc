get_magic_quotes_runtime

(PHP 4, PHP 5, PHP 7)

get_magic_quotes_runtime — 获取当前 magic_quotes_runtime 配置选项的激活状态

警告

本函数已自 PHP 7.4.0 起弃用，自 PHP 8.0.0 起移除。强烈建议不要依赖本函数。

说明

get_magic_quotes_runtime(): false

始终返回 false。

参数

此函数没有参数。

返回值

始终返回 false。

更新日志

版本 说明
7.4.0 弃用此函数。
