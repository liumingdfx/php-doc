finfo_close

(PHP >= 5.3.0, PHP 7, PHP 8, PECL fileinfo >= 0.1.0)

finfo_close — 关闭 finfo 实例

说明

finfo_close(finfo $finfo): bool

关闭 finfo_open() 函数所返回的 fileinfo 实例。

参数

finfo

经 finfo_open() 返回的 finfo 实例。

返回值

成功时返回 true， 或者在失败时返回 false。

更新日志

版本 说明
8.1.0 finfo 参数现在接受 finfo 实例，之前接受 resource。
