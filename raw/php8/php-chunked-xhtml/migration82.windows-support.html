Windows 支持

核心

Windows 特定错误消息不在本地化，而始终以英文显示以更好的匹配 PHP 错误消息。

新增初步且高度实验性的 ARM64 构建支持。

OCI8

因为无论如何都不再支持对 Oracle Client 10g 进行构建，所以已删除配置项 --with-oci8。仍然支持 --with-oci8-11g、--with-oci8-12c 和 --with-oci8-19 配置项。

Zip

Zip 扩展已升级到版本 1.21.0，且默认构建为共享库（DLL）。
